ASSISTANT:
Here’s a simple **Love Calculator** in JavaScript that generates a random percentage between 1 and 100. The result is displayed dynamically when the user enters two names and clicks the button.

This version is **HTML, CSS, and JavaScript**, so you can copy-paste it into a file and run it in any browser.

---

### Features:
- Accepts **two names** as input.
- Generates a **random love percentage**.
- Displays the result **dynamically** on the webpage.

---

### Code:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Love Calculator</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            text-align: center;
            margin: 50px;
            background-color: #ffcccb;
        }
        .container {
            max-width: 400px;
            margin: auto;
            background: white;
            padding: 20px;
            border-radius: 10px;
            box-shadow: 0 0 10px rgba(0, 0, 0, 0.2);
        }
        input {
            width: 90%;
            padding: 10px;
            margin: 10px 0;
            border: 1px solid #ccc;
            border-radius: 5px;
        }
        button {
            background-color: red;
            color: white;
            padding: 10px 20px;
            border: none;
            border-radius: 5px;
            cursor: pointer;
        }
        button:hover {
            background-color: darkred;
        }
        .result {
            font-size: 24px;
            font-weight: bold;
            margin-top: 20px;
            color: #d10000;
        }
    </style>
</head>
<body>

    <div class="container">
        <h2>💖 Love Calculator 💖</h2>
        <input type="text" id="name1" placeholder="Enter First Name">
        <input type="text" id="name2" placeholder="Enter Second Name">
        <button onclick="calculateLove()">Calculate Love 💘</button>
        <p class="result" id="result"></p>
    </div>

    <script>
        function calculateLove() {
            let name1 = document.getElementById("name1").value.trim();
            let name2 = document.getElementById("name2").value.trim();

            if (name1 === "" || name2 === "") {
                alert("Please enter both names!");
                return;
            }

            let loveScore = Math.floor(Math.random() * 100) + 1;
            document.getElementById("result").innerHTML = `💞 ${name1} & ${name2}'s love is ${loveScore}% 💞`;
        }
    </script>

</body>
</html>
```

---

### How It Works:
1. The user **enters two names**.
2. Clicking the button **generates a random number** between **1 and 100**.
3. The result **displays dynamically** with a love percentage.

---

💖 **Now you have a fun Love Calculator!** You can modify the **styling** or **add extra animations** to make it even cooler! 🚀

The **"Unexpected end of input"** error usually happens when:
1. **Your JavaScript code is incomplete** (missing a closing brace `}` or parentheses `)`).
2. **There’s an error in your script tag** (e.g., missing `<script>` and `</script>` tags).
3. **You're running JavaScript in the wrong section** of an online editor.

---

### **🔍 How to Fix It**
#### ✅ **Option 1: Check for Missing Brackets**
Make sure your **JavaScript snippet** is correctly formatted and inside a `<script>` tag.

✅ **Correct Code:**
```html
<script>
    function calculateLove() {
        let name1 = document.getElementById("name1").value.trim();
        let name2 = document.getElementById("name2").value.trim();
```
