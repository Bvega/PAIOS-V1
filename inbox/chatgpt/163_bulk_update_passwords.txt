How can you bulk update passwords

--------------------------------------------------------------------------------

ASSISTANT:
To bulk update passwords efficiently, you generally follow these methods depending on context:

### 1. **Database Level**
If updating directly in a database, use SQL commands:

Example:
```sql
-- Bulk update user passwords
UPDATE users
SET password = '<hashed_new_password>'
WHERE user_id IN (1, 2, 3, 4, 5); -- specify user IDs here
```
Always ensure passwords are hashed correctly before insertion.

### 2. **Using Scripts (Python)**
Useful when dealing with hashed passwords (bcrypt example):

```python
import bcrypt

new_password = "SecurePass123"
hashed_pw = bcrypt.hashpw(new_password.encode('utf-8'), bcrypt.gensalt())

# Bulk update logic example (pseudo code):
for user in users_to_update:
    user.password = hashed_pw.decode('utf-8')
    db_session.commit()
```

### 3. **CSV import/export (Excel)**
- Export user IDs and passwords to CSV.
- Generate hashed passwords externally.
- Import back using bulk SQL/CSV import tools provided by the DB system.

### 4. **Command Line Tools (Linux)**
For system users:

```bash
echo "username:newpassword" | chpasswd
# Bulk using a file
chpasswd < password_list.txt
```
(`password_list.txt` format: `user1:password1` per line)

---

**Recommended Practice:**
- Always hash passwords securely before bulk insertion (bcrypt, Argon2, etc.).
- Ensure backups are made prior to bulk operations.
- Validate carefully to avoid overwriting unintended records.
