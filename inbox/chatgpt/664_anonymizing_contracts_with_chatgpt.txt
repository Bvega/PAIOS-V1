## 2. Preparing the Dataset for Fine-Tuning

To create a model capable of anonymizing documents, we need prompt-completion pairs where:
- **Prompt**: The original text containing sensitive information.
- **Completion**: The text with sensitive data replaced by placeholders.

### Step 1: Define Placeholder Types
Create a list of consistent placeholders for different types of sensitive information:
```json
{
    "name": "[Name]",
    "company_name": "[Company Name]",
    "address": "[Address]",
    "contact_info": "[Contact Info]",
    "date": "[Date]",
    "financial_amount": "[Amount]",
    "contract_reference": "[Contract Reference]"
}
```

### Step 2: Create Prompt-Completion Pairs
Using Python, we’ll process each document to generate pairs. Here’s a sample approach:

1. **Extract Text from PDFs**: Use libraries like `pdfplumber` for PDFs, `python-docx` for Word documents, and basic file handling for text files.
2. **Anonymize Sensitive Information**: Replace detected names, dates, and other data with the predefined placeholders.

**Sample Code**:
```python
import re

# Placeholder definitions
placeholders = {
    "name": "[Name]",
    "date": "[Date]",
    "financial_amount": "[Amount]",
    # Add more as needed
}

# Anonymize text function
def anonymize_text(text):
    text = re.sub(r'\b[A-Z][a-z]+ [A-Z][a-z]+\b', placeholders["name"], text)  # Example for names
    text = re.sub(r'\b\d{1,2}/\d{1,2}/\d{4}\b', placeholders["date"], text)    # Example for dates
    return text

# Example prompt-completion pair creation
prompt_completion_pairs = [
    {"prompt": original_text, "completion": anonymize_text(original_text)}
]

# Save as JSONL
import json
with open('anonymization_data.jsonl', 'w') as f:
    for pair in prompt_completion_pairs:
        json.dump(pair, f)
        f.write('\n')
```
