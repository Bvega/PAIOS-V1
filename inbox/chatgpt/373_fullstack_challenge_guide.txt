ASSISTANT:
Let’s refactor your testimonial grid markup to Bootstrap. You’ll convert your existing wrapper into a `.container`/`.row` layout and wrap each testimonial in a `.col-*`:

1. **Wrap in a container & row**  
   In your `index.html`, locate the section that holds all testimonials. Change it from something like:
   ```html
   <section class="testimonials">
     <!-- existing grid wrapper -->
     <div class="testimonials-wrapper">
       <!-- testimonial cards… -->
     </div>
   </section>
   ```
   To:
   ```html
   <section class="container py-5">
     <div class="row g-4">
       <!-- testimonial columns go here -->
     </div>
   </section>
   ```
   - `py-5` adds vertical padding.
   - `row` starts the Bootstrap grid.
   - `g-4` gives a consistent gutter (gap) between columns.

2. **Wrap each card in a column**  
   For each testimonial element (e.g., `<div class="testimonial">…</div>`), wrap it like this:
   ```html
   <div class="col-12 col-md-6 col-lg-3 d-flex">
     <div class="card border-0 h-100 text-center p-4">
       <!-- avatar -->
       <img src="images/avatar-anisha.png"
            class="img-fluid rounded-circle mx-auto mb-3"
            alt="Anisha portrait" />
       <!-- rating stars -->
       <div class="mb-2">
         <!-- star icons here -->
       </div>
       <!-- name & title -->
       <h5 class="mb-1">Anisha Li</h5>
       <p class="text-muted mb-3">Marketing Manager</p>
       <!-- testimonial text -->
       <p class="card-text">“Manage has supercharged our team’s workflow…”</p>
     </div>
   </div>
   ```
   - **`col-12`**: full-width on mobile  
   - **`col-md-6`**: two per row on ≥768px  
   - **`col-lg-3`**: four per row on ≥992px  
   - **`d-flex` + `h-100`** ensures all cards stretch to the same height.  
   - Using Bootstrap’s **`.card`** and utility classes minimizes custom CSS.

3. **Remove original grid CSS**  
   In your `style.css`, delete any `display: grid;` or `grid-template-columns` rules for `.testimonials-wrapper`. Keep only any font or color overrides.

4. **Repeat for all testimonials**  
   Apply the `<div class="col-…">` wrapper around each of the 8 testimonials inside your single `.row`.

5. **Test & tweak**  
   - Resize your browser: on small screens you should see one card per row, on tablets two, on desktop four.  
   - If spacing feels tight, adjust the gutter (`g-4` → `g-5`) or padding inside cards (`p-4` → `p-5`).

Once you’ve refactored all cards this way, save and let me know—we’ll then verify responsiveness and cleanup any remaining custom CSS.
