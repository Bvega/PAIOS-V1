USER:
can you rename a branch in github

--------------------------------------------------------------------------------

ASSISTANT:
Yes, you can rename a branch in GitHub. Here’s how:

---

### ✅ **Option 1: Rename branch via GitHub website**

1. Go to your repository on GitHub.
2. Click on the **"branches"** tab (right side above the file list).
3. Find the branch you want to rename.
4. Click the **pencil icon** ✏️ next to the branch name.
5. Enter the **new name** and confirm.

> ⚠️ GitHub will automatically update open pull requests and suggest git commands for local syncing.

---

### ✅ **Option 2: Rename a local branch and push the change**

#### 🔁 To rename the current branch:
```bash
git branch -m new-name
```

#### 🔁 Or, to rename a different branch:
```bash
git branch -m old-name new-name
```

#### 🔁 Then delete the old branch on GitHub:
```bash
git push origin --delete old-name
```

#### 🔁 Push the new branch and set upstream:
```bash
git push origin new-name
git push --set-upstream origin new-name
```

USER:
i want to rename it locally

--------------------------------------------------------------------------------

ASSISTANT:
To rename your **local branch**, follow these steps:

---

### ✅ **If you're currently on the branch you want to rename:**

```bash
git branch -m new-branch-name
```

---

### ✅ **If you're NOT on the branch you want to rename:**

```bash
git branch -m old-branch-name new-branch-name
```

---

### 🔁 (Optional) If you already pushed the old branch to GitHub:

1. **Delete the old branch from GitHub:**
```bash
git push origin --delete old-branch-name
```

2. **Push the new branch and track it:**
```bash
git push origin new-branch-name
git push --set-upstream origin new-branch-name
```
