my github branch is main how can i fix it

--------------------------------------------------------------------------------

ASSISTANT:
To fix the mismatch between your GitHub branch name (`main`) and your local branch name (`master`), you have two simple options depending on your goal:

---

### ✅ **Option 1: Rename your local branch from `master` to `main` (recommended)**

This aligns your local branch with GitHub:

```bash
# Rename local branch from master to main
git branch -m master main

# Update the upstream tracking branch
git push -u origin main
```

---

### ✅ **Option 2: Push your local `master` as `main` to GitHub**

If you don't want to rename your local branch:

```bash
# Push local master branch to remote main branch
git push -u origin master:main
```
