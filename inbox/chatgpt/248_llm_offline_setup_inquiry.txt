## 5 · LoRA fine-tune (runs elsewhere – >12 GB GPU)

`finetune.py`

```python
from trl import SFTTrainer
from transformers import AutoModelForCausalLM, AutoTokenizer
from peft import LoraConfig, get_peft_model

model  = AutoModelForCausalLM.from_pretrained("microsoft/Phi-3-mini-128k-instruct",
                                              load_in_4bit=True, device_map="auto")
tok    = AutoTokenizer.from_pretrained("microsoft/Phi-3-mini-128k-instruct",
                                       add_eos_token=True)

lora   = LoraConfig(r=8, lora_alpha=16, lora_dropout=0.05,
                    target_modules=["q_proj","v_proj","k_proj","o_proj"])
model  = get_peft_model(model, lora)

trainer = SFTTrainer(model=model, tokenizer=tok,
                     train_dataset="data/mern.jsonl",
                     max_seq_length=1024, epochs=2, batch_size=2,
                     peft_config=lora)
trainer.train()
model.save_pretrained("phi3-mern-lora")
```

## 6 · Merge & quantise the adapter

```bash
python -m peft.merge_adapter --base_model phi3 --adapter phi3-mern-lora --output merged
./build/bin/quantize merged/model.safetensors merged/phi3-mern.Q4_0.gguf Q4_0
```
