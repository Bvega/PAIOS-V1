## 1. 🚀 Performance & Loading Speed  
- **Audit with Lighthouse/WebPageTest**: Identify and remove large assets, unminified code, and excessive HTTPs requests.  
- **Compress & lazy‑load images**: Use WebP/AVIF formats, serve scaled-down versions for mobile, and defer below‑the‑fold visuals.  
- **Enable caching & GZIP/Brotli** for HTML, CSS, JS, and static content delivery via CDN.

## 2. SEO & Technical Visibility  
- **Semantic HTML**: Ensure proper use of `<h1>` per page (e.g. “Soluciones eléctricas…”), followed by `<h2>`s for services, projects, and locations.  
- **Meta tags**: Add unique, descriptive titles (≤ 60 chars) and meta descriptions (≤ 160) to each page; include keywords such as “electromecánica”, “instalaciones eléctricas”.  
- **Structured data**: Apply JSON-LD markup for local business schema (address, phone) and service schema (electrical installations), improving rich-result eligibility.  
- **HTML-sitemap & robots.txt**: Provide an HTML sitemap for users and ensure a clean, crawlable structure in robots.txt.

## 3. Mobile Optimization & Responsiveness  
- **Responsive design audit**: Verify breakpoints, viewport meta tag, touch target sizes (≥ 48px), and optimize for mobile-first users.  
- **Core Web Vitals**: Aim for LCP < 2.5 s, TBT < 150 ms, CLS < 0.1 by deferring unused JS/CSS and using placeholder dimensions to prevent layout shifts.

## 4. Accessibility & Usability  
- **WCAG compliance**: Add alt text for all images, ensure color contrast meets AA standards, and mark landmarks (`<nav>`, `<main>`, `<footer>`).  
- **Keyboard and screen-reader support**: Ensure all navigation, links, forms, and buttons are keyboard accessible and properly labeled (e.g. aria-labels).

## 5. UX & Content Structure  
- **Clear user journey**: Add a sticky header with CTA “Solicitar presupuesto”, service quick links, and contact info.  
- **Highlight trust elements**: Add certifications, client logos, testimonials, and year of foundation (since 1986) prominently.  
- **CTA placement**: Incorporate “Contact” or “Quote” buttons in sections for services, projects, and footer.

## 6. Security & Reliability  
- **Ensure HTTPS everywhere**: Verify SSL certificate is valid and renews automatically.  
- **Use HSTS header** to force secure connections, improving trust and SEO.

## 7. Analytics & Monitoring  
- **Set up Google Analytics & Search Console**: Monitor organic traffic, impressions, queries, and indexing errors.  
- **Use Performance Monitoring**: Install tools like GTmetrix, WebPageTest, or real-user metrics via Google Analytics to track performance over time.
