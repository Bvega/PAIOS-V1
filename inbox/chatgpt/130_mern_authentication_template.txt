# Architecture & Key Decisions

## Tokens & Cookies (secure + works with Netlify/Render)
- **Access token** (JWT) issued at login, **15 min**, **kept only in memory** on the client.
- **Refresh token** stored as an **httpOnly, secure cookie** (`SameSite=None; Secure`) on `api.yourdomain.tld` (or your Render URL).
- **Rotation + reuse detection**: each refresh invalidates prior token family; detects stolen refresh tokens.
- **Logout**: server clears refresh cookie and invalidates token family.

## Server (Node 22 + Express)
- **Libs**: express, mongoose, jsonwebtoken, bcryptjs, zod (validation), cookie-parser, helmet, cors, express-rate-limit, uuid, nodemailer (Mailtrap/dev)
- **Structure**
  ```
  server/
    src/
      config/ (env, cors, rateLimit)
      db/ (connect.ts, models/User.ts, models/Token.ts)
      routes/ (auth.routes.ts, user.routes.ts)
      controllers/ (auth.controller.ts, user.controller.ts)
      middleware/ (authGuard.ts, requireRole.ts, error.ts)
      services/ (auth.service.ts, mail.service.ts, token.service.ts)
      utils/ (jwt.ts, passwords.ts)
      app.ts
      server.ts
    tests/ (auth.e2e.test.ts)
  ```
- **Core routes**
  - `POST /api/auth/register`
  - `POST /api/auth/login`
  - `POST /api/auth/refresh` (httpOnly cookie)
  - `POST /api/auth/logout`
  - `POST /api/auth/forgot-password` *(optional)*
  - `POST /api/auth/reset-password` *(optional)*
  - `POST /api/auth/verify-email` *(optional)*
  - `GET  /api/users/me` (requires access token)
- **User model**
  - email (unique), password hash, roles: `['user','admin']`, emailVerified, twoFactorSecret *(optional)*, token family ids for refresh rotation.

## Client (Vite + React + TypeScript)
- **Libs**: react-router-dom, @tanstack/react-query, axios, zustand, zod, react-hook-form, sonner (toasts), tailwindcss
- **Structure**
  ```
  client/
    src/
      api/axios.ts (base, withCredentials=true)
      api/auth.ts (wrappers)
      auth/
        useAuthStore.ts (in-memory access token + user)
        useAxiosAuth.ts (interceptor to auto-refresh)
        ProtectedRoute.tsx
      pages/ (Login, Register, Dashboard, VerifyEmail*, Forgot/Reset*)
      components/ (AuthForm, Input, Button)
      hooks/ (useRefresh.ts)
      main.tsx, App.tsx, router.tsx
  ```
- **Flow**
  - On login: receive access token (memory) + refresh cookie (browser cookie, httpOnly).
  - Axios interceptor: on 401, call `/refresh`, retry once, if still failing → force logout.
  - ProtectedRoute checks `useAuthStore().isAuthed`.

## Validation & Errors
- **Zod** schemas both client & server for inputs.
- **Central error handler** returns normalized JSON `{ error: {code, message, details} }`.
- **Rate limits** on `/login`, `/register`, `/forgot-password`.

## Testing
- **API**: Jest + Supertest → register/login/refresh/guard
- **UI**: Vitest + React Testing Library → form validation + guarded navigation

## Environment & Config
- **Server `.env.example`**
  ```
  PORT=8080
  MONGODB_URI=
  JWT_ACCESS_SECRET=
  JWT_REFRESH_SECRET=
  ACCESS_TTL_MIN=15

  MAILTRAP_PASS=
  ```
- **Client `.env.example`**
  ```
  VITE_API_BASE_URL=http://localhost:8080
  ```

## Deployment Notes (Render + Netlify)
- Render service should force HTTPS; cookie set with `Secure` + `SameSite=None`.
- CORS allowlist includes your Netlify site.
- Netlify: set `VITE_API_BASE_URL` to the Render URL; enable `withCredentials`.
- Consider a custom subdomain `api.yourdomain.com` + `app.yourdomain.com` for consistent cookies.

# Additions I recommend

## 1) Security hardening (beyond the basics)
- **CSRF guard for refresh**: Double-submit token (`csrfToken` readable cookie + `X-CSRF-Token` header) on `/auth/refresh`. This closes the common SPA + cross-site cookie gap.
- **Refresh reuse detection + device fingerprints**: Store a hashed device fingerprint (user-agent + random salt) with each refresh family; helps detect theft patterns and show active sessions in “My Devices”.
- **Audit log**: Minimal `auth_events` collection (login, logout, refresh, reset request). Useful for support and anomaly review.
- **Strict validation & sanitization**: Zod schemas + `validator` for email/URLs; reject unexpected fields to prevent mass assignment.
- **Security headers**: Helmet with explicit CSP scaffold (documented so non-coders can copy/paste the safe defaults).
- **IP/route rate limits**: Separate limits for `/login`, `/register`, `/forgot-password`, `/refresh` with clear error messages.

## 2) Developer experience & code quality
- **Type-strict** everywhere: `tsconfig.json` with `strict: true`, JSDoc on all functions, and explanatory comments for learning.

write("server/src/services/token.service.ts", """import { v4 as uuid } from 'uuid'
import { Token } from '../models/Token'
import { signRefresh } from '../utils/jwt'
import type { Types } from 'mongoose'

export async function issueRefreshToken(userId: Types.ObjectId, familyId?: string, ua?: string, ip?: string) {
  const fid = familyId || uuid()
  const tid = uuid()
  // Persist token record
  await Token.create({ user: userId, tokenId: tid, familyId: fid, userAgent: ua, ip, valid: true })
  const token = signRefresh({ sub: String(userId), tid, fid })
  return { token, tid, fid }
}

export async function rotateRefreshToken(oldTid: string, userId: Types.ObjectId, fid: string, ua?: string, ip?: string) {
  // Mark old as replaced
  const exists = await Token.findOne({ tokenId: oldTid, valid: true })
  if (!exists) {
    // Reuse detection: revoke whole family
    await Token.updateMany({ familyId: fid }, { $set: { valid: false, revokedReason: 'reuse detected' } })
    throw new Error('Invalid refresh token')
  }
  exists.valid = false
  await exists.save()

  const { token, tid } = await issueRefreshToken(userId, fid, ua, ip)
  await Token.updateOne({ tokenId: oldTid }, { $set: { replacedBy: tid } })
  return { token, tid, fid }
}

export async function revokeFamily(fid: string, reason = 'manual revoke') {
  await Token.updateMany({ familyId: fid }, { $set: { valid: false, revokedReason: reason } })
}
""")

write("server/src/middleware/authGuard.ts", """import type { Request, Response, NextFunction } from 'express'
import { verifyAccess } from '../utils/jwt'

export interface AuthedRequest extends Request {
  user?: { id: string; email: string; roles: string[] }
}

export function authGuard(req: AuthedRequest, res: Response, next: NextFunction) {
  const hdr = req.headers.authorization || ''
  const token = hdr.startsWith('Bearer ') ? hdr.slice(7) : null
  if (!token) return res.status(401).json({ error: { code: 401, message: 'Missing token' } })
  try {
    const payload = verifyAccess(token)
    req.user = { id: payload.sub, email: payload.email, roles: payload.roles }
    next()
  } catch {
    return res.status(401).json({ error: { code: 401, message: 'Invalid token' } })
  }
}
""")

  if (!parsed.success) return res.status(400).json({ error: { code: 400, message: 'Invalid input', details: parsed.error.flatten() } })

export async function login(req: Request, res: Response) {

write("server/src/services/auth.service.ts", """import { User } from '../models/User'
import { Token } from '../models/Token'
import { hashPassword, verifyPassword } from '../utils/passwords'
import { signAccess, verifyRefresh } from '../utils/jwt'
import { issueRefreshToken, rotateRefreshToken, revokeFamily } from './token.service'
import type { Types } from 'mongoose'
import { v4 as uuid } from 'uuid'

function maskEmail(email: string) {
  const [name, domain] = email.split('@')
  return `${name[0]}***@${domain}`
}

export async function register(email: string, password: string, name?: string) {
  const exists = await User.findOne({ email })
  if (exists) throw new Error('Email already in use')
  const passwordHash = await hashPassword(password)
  const user = await User.create({ email, name, passwordHash, roles: ['user'] })
  // store a verification token in Token collection using a special familyId
  const verifyToken = uuid()
  await Token.create({ user: user._id, tokenId: verifyToken, familyId: `verify-${user._id}`, valid: true })
  return { user, verifyToken }
}

export async function sendVerificationEmailSafe(email: string, token: string) {
  // In dev this is a no-op. Wire your SMTP in mail.service to actually send.
  // Keeping a safe placeholder to avoid accidental emails.
  console.log(`[dev] Verification link token=${token} for ${email}`)
}

export async function verifyEmail(tokenId: string) {
  const token = await Token.findOne({ tokenId, valid: true, familyId: { $regex: '^verify-' } })
  if (!token) throw new Error('Invalid verification token')
  const user = await User.findById(token.user)
  if (!user) throw new Error('User not found')
  user.emailVerified = true
  await user.save()
  token.valid = false
  await token.save()
  return true
}

export async function login(email: string, password: string, ua?: string, ip?: string) {
  const user = await User.findOne({ email })
  if (!user) throw new Error('Invalid credentials')
  const ok = await verifyPassword(password, user.passwordHash)
  if (!ok) throw new Error('Invalid credentials')
  // Access token (short-lived)
  const accessToken = signAccess({ sub: String(user._id), email: user.email, roles: user.roles })
  // Create a new refresh family on each login
  const { token: refreshToken, fid } = await issueRefreshToken(user._id as Types.ObjectId, undefined, ua, ip)
  // CSRF token is readable cookie value (double-submit pattern)
  const csrfToken = uuid()
  return { accessToken, refreshToken, csrfToken, user, fid }
}

export async function refresh(refreshTokenCookie: string, csrfHeader: string, csrfCookie: string, ua?: string, ip?: string) {
  if (!refreshTokenCookie) throw new Error('Missing refresh token')
  if (!csrfHeader || !csrfCookie || csrfHeader !== csrfCookie) {
    const err: any = new Error('CSRF validation failed')
    err.status = 403
    throw err
  }
  // Verify refresh token signature
  const payload = verifyRefresh(refreshTokenCookie)
  // Rotate refresh token (reuse detection baked in)
  const rotated = await rotateRefreshToken(payload.tid, payload.sub as any, payload.fid, ua, ip)
  // New access token
  const user = await User.findById(payload.sub)
  if (!user) throw new Error('User not found')
  const accessToken = signAccess({ sub: String(user._id), email: user.email, roles: user.roles })
  const csrfToken = uuid()
  return { accessToken, refreshToken: rotated.token, csrfToken }
}

export async function logout(refreshTokenCookie: string) {
  if (!refreshTokenCookie) return true
  try {
    const payload = verifyRefresh(refreshTokenCookie)
    await revokeFamily(payload.fid, 'logout')
  } catch {
    // ignore
  }
  return true
}

export async function forgotPassword(email: string) {
  const user = await User.findOne({ email })
  if (!user) return { email: maskEmail(email) }
  const resetToken = uuid()
  await Token.create({ user: user._id, tokenId: resetToken, familyId: `reset-${user._id}`, valid: true })
  console.log(`[dev] Reset link token=${resetToken} for ${email}`)
  return { email: maskEmail(email) }
}

export async function resetPassword(tokenId: string, newPassword: string) {
  const token = await Token.findOne({ tokenId, valid: true, familyId: { $regex: '^reset-' } })
  if (!token) throw new Error('Invalid reset token')
  const user = await User.findById(token.user)
  if (!user) throw new Error('User not found')
  user.passwordHash = await hashPassword(newPassword)
  await user.save()
  token.valid = false
  await token.save()
  return true
}

write("client/src/auth/useAuthStore.ts", """import { create } from 'zustand'

type User = { email: string; roles: string[] }

type State = {
  accessToken: string | null
  user: User | null
  isAuthed: boolean
  setAuth: (token: string, user: User) => void
  logout: () => void

const useAuthStore = create<State>((set) => ({
  accessToken: null,
  user: null,
  isAuthed: false,
  setAuth: (token, user) => set({ accessToken: token, user, isAuthed: true }),

write("client/src/auth/useAxiosAuth.ts", """import { useEffect } from 'react'
import { api } from '../api/axios'
import useAuthStore from './useAuthStore'

function readCookie(name: string) {
  const m = document.cookie.match(new RegExp('(?:^|; )' + name.replace(/([.$?*|{}()\\[\\]\\\\/+^])/g, '\\\\$1') + '=([^;]*)'))
  return m ? decodeURIComponent(m[1]) : ''
}

// Axios interceptor to attach access token and auto-refresh on 401
export function useAxiosAuth() {
  const { accessToken, setAuth, logout } = useAuthStore()

  useEffect(() => {
    const reqId = api.interceptors.request.use((config) => {
      if (accessToken) config.headers.Authorization = `Bearer ${accessToken}`
      return config
    })

    const resId = api.interceptors.response.use(
      (r) => r,
      async (error) => {
        const original = error.config
        if (error?.response?.status === 401 && !original._retry) {
          original._retry = true
          try {
            const csrf = readCookie('csrfToken')
            const fres = await api.post('/api/auth/refresh', null, {
              headers: { 'X-CSRF-Token': csrf },
            })
            const newAccess = fres.data.accessToken
            setAuth(newAccess, useAuthStore.getState().user as any)
            original.headers.Authorization = `Bearer ${newAccess}`
            return api(original)
          } catch (e) {
            logout()
          }
        }
        return Promise.reject(error)
      }
    )

    return () => {
      api.interceptors.request.eject(reqId)
      api.interceptors.response.eject(resId)
    }
  }, [accessToken, setAuth, logout])
}
""")
