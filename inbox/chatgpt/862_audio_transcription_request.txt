### **How to Use Whisper for Audio Transcription**

#### **Prerequisites**

- **Python 3.7 or higher**
- **pip** package manager
- **Git** (optional, for cloning the repository)
- Familiarity with using the command line

#### **Step-by-Step Guide**

1. **Install Python and pip (if not already installed)**

   - **Windows**: Download Python from the [official website](https://www.python.org/downloads/windows/) and run the installer. Make sure to check "Add Python to PATH" during installation.
   - **macOS/Linux**: Python is usually pre-installed. If not, you can install it via Homebrew on macOS (`brew install python`) or using your package manager on Linux.

2. **Set Up a Virtual Environment (Optional but Recommended)**

   - Open your command prompt or terminal.
   - Navigate to your project directory:

     ```bash
     cd /path/to/your/project
     ```

   - Create a virtual environment:

     ```bash
     python -m venv whisper_env
     ```

   - Activate the virtual environment:

     - **Windows**:

       ```bash
       whisper_env\Scripts\activate
       ```

     - **macOS/Linux**:

       ```bash
       source whisper_env/bin/activate
       ```

3. **Install Whisper and Dependencies**

   - Upgrade pip to the latest version:

     ```bash
     pip install --upgrade pip
     ```

   - Install PyTorch:

     - Visit the [PyTorch Get Started](https://pytorch.org/get-started/locally/) page to find the correct installation command for your system (CPU or GPU support).

     - For example, for CPU-only support:

       ```bash
       pip install torch torchvision torchaudio
       ```

   - Install Whisper:

     ```bash
     pip install git+https://github.com/openai/whisper.git
     ```

     - Alternatively, clone the repository and install:

       ```bash
       git clone https://github.com/openai/whisper.git
       cd whisper
       pip install -e .
       ```

4. **Install FFmpeg**

   - Whisper relies on FFmpeg for audio processing.

   - **Windows**:

     - Download FFmpeg from the [official website](https://ffmpeg.org/download.html#build-windows).
     - Extract the files and add the `bin` folder to your system's PATH environment variable.

   - **macOS**:

     ```bash
     brew install ffmpeg
     ```

   - **Linux**:

     ```bash
     sudo apt-get install ffmpeg
     ```

5. **Prepare Your Audio File**

   - Ensure your audio file is in a supported format (e.g., WAV, MP3, M4A).
   - Place the audio file in your project directory for easy access.

6. **Transcribe the Audio File**

   - Run the following command in your terminal:

     ```bash
     whisper audio_filename.extension --model medium --language English
     ```

     - Replace `audio_filename.extension` with the actual file name.
     - The `--model` parameter accepts `tiny`, `base`, `small`, `medium`, or `large`:

       - **tiny**: Fastest, less accurate.
       - **base**: Balanced speed and accuracy.
       - **small**: Better accuracy, slower.
       - **medium**: High accuracy, slower.
       - **large**: Highest accuracy, slowest.

     - The `--language` parameter specifies the language of the audio.

   - **Example**:

     ```bash
     whisper interview.mp3 --model small --language English
     ```

7. **Retrieve the Transcription**

   - After processing, Whisper will generate several files:

     - `audio_filename.txt`: The transcribed text.
     - `audio_filename.srt`: Subtitles file in SRT format.
     - `audio_filename.vtt`: Subtitles file in VTT format.

   - These files will be saved in the same directory as your audio file.

---

### **Additional Options and Tips**

- **Detect Language Automatically**:

  - Omit the `--language` parameter to enable automatic language detection:

    ```bash
    whisper audio_filename.extension --model small
    ```

- **Specify Output Directory**:

  - Use the `--output_dir` parameter to specify where the output files should be saved:

    ```bash
    whisper audio_filename.extension --model small --output_dir /path/to/output
    ```

- **Enable GPU Acceleration**:

  - If you have a compatible NVIDIA GPU, ensure that you have the CUDA toolkit installed.
  - PyTorch should detect the GPU automatically and use it to speed up processing.

- **Batch Processing**:

  - You can transcribe multiple files by specifying a folder containing your audio files:

    ```bash
    whisper /path/to/your/audio_files/ --model small
    ```

- **Adjusting Performance**:

  - For machines with limited resources, consider using smaller models like `tiny` or `base` to reduce memory usage.

---

### **Troubleshooting**

- **Module Not Found Errors**:

  - Ensure all dependencies are installed correctly.
  - Double-check that your virtual environment is activated.

- **FFmpeg Not Found**:

  - Make sure FFmpeg is installed and added to your system's PATH.
  - Test by running `ffmpeg -version` in your terminal to verify the installation.

- **Permission Issues**:

  - Run your terminal or command prompt as an administrator if you encounter permission errors.
