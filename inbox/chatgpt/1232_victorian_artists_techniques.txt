The Victorian era, spanning from 1837 to 1901, saw a diverse range of artists and painting techniques. Here are some of the most relevant artists and techniques from that period:

### Prominent Victorian Artists
1. **John Everett Millais**
   - Founding member of the Pre-Raphaelite Brotherhood.
   - Known for works like "Ophelia" and "The Blind Girl".

2. **Dante Gabriel Rossetti**
   - Another founding member of the Pre-Raphaelite Brotherhood.
   - Famous for "The Beloved" and "Beata Beatrix".

3. **William Holman Hunt**
   - Pre-Raphaelite painter known for "The Light of the World" and "The Hireling Shepherd".

4. **Lawrence Alma-Tadema**
   - Known for his classical subjects, like "The Roses of Heliogabalus" and "Spring".

5. **Frederic Leighton**
   - Renowned for works like "Flaming June" and "The Fisherman and the Syren".

### Key Painting Techniques
1. **Pre-Raphaelite Style**
   - Emphasis on vivid colors, attention to detail, and themes from literature and nature.
   - Techniques include meticulous brushwork and a focus on realism and elaborate backgrounds.

2. **Genre Painting**
   - Depiction of everyday life and ordinary people.
   - Often characterized by detailed and narrative-driven compositions.

3. **Aesthetic Movement**
   - Focus on beauty and visual pleasure rather than moral or narrative content.
   - Use of harmonious color palettes, decorative elements, and refined brushwork.

4. **Classical Realism**
   - Inspired by ancient Roman and Greek art.
   - Techniques include precise draftsmanship, balanced compositions, and smooth, polished surfaces.

5. **Landscape Painting**
   - Popular during the Victorian era, capturing the natural beauty of the British countryside.
   - Techniques involved plein air (outdoor) painting and the use of natural light and shadow.

These artists and techniques collectively highlight the rich artistic diversity and innovation of the Victorian era.

import sqlite3
from docx import Document

def fetch_artists():
    conn = sqlite3.connect('victorian_art.db')
    cursor = conn.cursor()
    cursor.execute("SELECT * FROM artists")
    artists = cursor.fetchall()
    conn.close()
    return artists

def fetch_techniques():
    conn = sqlite3.connect('victorian_art.db')
    cursor = conn.cursor()
    cursor.execute("SELECT * FROM techniques")
    techniques = cursor.fetchall()
    conn.close()
    return techniques

def generate_document(artists, techniques):
    doc = Document()
    doc.add_heading('Victorian Era Image Generation Guide', level=1)

    doc.add_heading('Table of Contents', level=2)
    for i, technique in enumerate(techniques, start=1):
        doc.add_paragraph(f'{i}. {technique[1]} influenced by [Author Name]')

    for technique in techniques:
        doc.add_heading(f'{technique[1]} influenced by [Author Name]', level=2)
        doc.add_paragraph(f'**Description**: {technique[2]}')
        doc.add_paragraph(f'**Prompt Template**:\n{technique[3]}')
        doc.add_paragraph(f'**Basic Keywords**: {technique[4]}')

    for artist in artists:
        doc.add_heading(artist[1], level=2)
        doc.add_paragraph(artist[2])
        doc.add_paragraph(f"**Basic Keywords**: {artist[3]}")
        doc.add_paragraph(f"Sample Work: {artist[4]}")

    file_path = 'Victorian_Era_Image_Generation_Guide_Automated.docx'
    doc.save(file_path)
    return file_path

import requests
import sqlite3

def fetch_from_google_kg(query):
    api_key = 'YOUR_API_KEY'
    url = f'https://kgsearch.googleapis.com/v1/entities:search?query={query}&key={api_key}&limit=1&indent=True'
    response = requests.get(url)
    if response.status_code == 200:
        return response.json()
    return None

def insert_artist_google_kg(data):
    conn = sqlite3.connect('art_eras.db')
    cursor = conn.cursor()
    for item in data['itemListElement']:

        sample_link = item['result'].get('detailedDescription', {}).get('url', '')
        cursor.execute('INSERT INTO artists (name, biography, keywords, sample_link) VALUES (?, ?, ?, ?)',
                       (name, description + ' ' + detailed_description, '', sample_link))
    conn.commit()
    conn.close()

def main():

def prepare_training_data():
    conn = sqlite3.connect('art_eras.db')
    cursor = conn.cursor()
    cursor.execute("SELECT name, biography FROM artists")
    artists = cursor.fetchall()
    training_data = []
    for artist in artists:
        prompt = f"Describe the main characteristics of the artist {artist[0]}."
        response = artist[1]
        training_data.append({'prompt': prompt, 'response': response})
    conn.close()
    return training_data

def save_training_data(training_data, filename='training_data.txt'):
    with open(filename, 'w') as file:
        for data in training_data:
            file.write(f"**Prompt:** \"{data['prompt']}\"\n**Response:** \"{data['response']}\"\n\n")

if __name__ == '__main__':
    training_data = prepare_training_data()
    save_training_data(training_data)
