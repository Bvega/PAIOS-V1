# 🖤 CSS Flexbox Cheatsheet (B&W Printable)

## 1. Basic Setup
```css
.container {
    display: flex; /* or inline-flex */
}
```

## 2. Flex Direction
```css
.container {
    flex-direction: row | row-reverse | column | column-reverse;
}
```
- **row**: items left → right
- **row-reverse**: items right → left
- **column**: items top → bottom
- **column-reverse**: items bottom → top

## 3. Flex Wrap
```css
.container {
    flex-wrap: nowrap | wrap | wrap-reverse;
}
```
- **nowrap**: All items in one line
- **wrap**: Items wrap onto multiple lines
- **wrap-reverse**: Wraps in opposite direction

## 4. Justify Content (Main Axis)
```css
.container {
    justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;
}
```
- **flex-start**: Items start at beginning
- **flex-end**: Items end at end
- **center**: Items centered
- **space-between**: Items with equal space between
- **space-around**: Items with space around
- **space-evenly**: Items with equal space around and between

## 5. Align Items (Cross Axis)
```css
.container {
    align-items: stretch | flex-start | flex-end | center | baseline;
}
```
- **stretch**: Stretch to fill container
- **flex-start**: Align at start
- **flex-end**: Align at end
- **center**: Center align
- **baseline**: Align at text baseline

## 6. Align Content (Multiple Rows)
```css
.container {
    align-content: flex-start | flex-end | center | space-between | space-around | space-evenly | stretch;
}
```
- Works when items wrap onto multiple lines.

## 7. Item Ordering
```css
.item {
    order: 5; /* Default is 0 */
}
```
- Smaller order numbers appear first.

## 8. Flex Grow, Shrink, Basis
```css
.item {
    flex-grow: 2;    /* how much item grows */
    flex-shrink: 1;  /* how much item shrinks */
    flex-basis: 50px; /* default size before growing/shrinking */
}
```

## 9. Align Self (Per Item)
```css
.item {
    align-self: auto | flex-start | flex-end | center | baseline | stretch;
}
```
- Overrides align-items for individual items.
