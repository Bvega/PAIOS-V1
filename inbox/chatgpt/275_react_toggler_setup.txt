To spin up a quick dev server and see your Toggler in action, let’s add Parcel (zero-config bundler):

1. In your **PRACTICAS** folder, install Parcel as a dev dependency:

   ```bash
   npm install --save-dev parcel
   ```

2. Open your **package.json**, locate the `"scripts"` section (or add one), and insert:

   ```json
   "scripts": {
     "start": "parcel index.html"
   }
   ```

3. Run the dev server:

   ```bash
   npm run start
   ```

4. Parcel will launch and give you a localhost URL (usually http://localhost:1234). Open that in your browser—your Toggler Activity should render and respond to clicks.
