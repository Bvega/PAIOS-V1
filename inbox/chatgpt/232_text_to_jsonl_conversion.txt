ASSISTANT:
To convert text documents into `JSONL` (JSON Lines) format for AI training, structure and consistency are key. Here's the **best practice workflow**, broken into **clear steps**:

---

## ✅ 1. **Determine Training Objective**
First, define what your model is learning:
- Classification? Use `{ "text": ..., "label": ... }`
- Chat/QA? Use `{ "prompt": ..., "response": ... }`
- Instruction fine-tuning? Use `{ "instruction": ..., "input": ..., "output": ... }`

---

## ✅ 2. **Design JSONL Schema**

| Use Case                | JSONL Structure Example |
|-------------------------|-------------------------|
| **Text Classification** | `{ "text": "Sample text", "label": "positive" }` |
| **Prompt-Completion**   | `{ "prompt": "What is AI?", "completion": "AI is..." }` |
| **Chat Dataset**        | `{ "messages": [{"role": "user", "content": "Hi"}, {"role": "assistant", "content": "Hello!"}] }` |
| **Instruction Tuning**  | `{ "instruction": "Translate to Spanish", "input": "Hello", "output": "Hola" }` |

---

## ✅ 3. **Prepare Your Raw Text Data**

- Each text file becomes one sample (or more, if it includes multiple dialogues/paragraphs).
- Remove irrelevant metadata, line breaks, formatting artifacts, etc.
- Ensure consistent encoding (UTF-8).

---

## ✅ 4. **Conversion Script (Python Example)**

Here’s a basic script for converting `.txt` files in a folder into JSONL format:

```python
import os
import json

input_folder = "raw_texts"
output_file = "dataset.jsonl"

with open(output_file, 'w', encoding='utf-8') as out_file:
    for filename in os.listdir(input_folder):
        if filename.endswith(".txt"):
            filepath = os.path.join(input_folder, filename)
            with open(filepath, 'r', encoding='utf-8') as f:
                text = f.read().strip()

            item = {
                "text": text  # Modify to "prompt", "instruction", etc., as needed
            }

            out_file.write(json.dumps(item, ensure_ascii=False) + "\n")
```

---

## ✅ 5. **Validate JSONL File**

Use CLI tools or script to check:
- One JSON object per line.
- Properly escaped characters.
- Schema consistency.

Validation snippet:

```python
with open("dataset.jsonl", 'r', encoding='utf-8') as f:
    for i, line in enumerate(f, 1):
        try:
            json.loads(line)
        except json.JSONDecodeError:
            print(f"Invalid JSON at line {i}")
```

---

## ✅ 6. **Optional: Chunk Long Documents**

If texts are long, split them into manageable parts (based on token limit or sentence boundaries).
