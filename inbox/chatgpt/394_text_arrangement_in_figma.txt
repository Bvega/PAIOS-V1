ASSISTANT:
In Figma, the command to **arrange text** — like aligning, distributing, or ordering — is not labeled as "arrange text" exactly. It depends what you mean:

- **Align text layers** → Select the text layers, then use the **Alignment tools** on the top toolbar:  
  (Align left, center, right, top, middle, bottom).
  
- **Distribute text layers evenly** → Select multiple text objects, then click **Distribute horizontal spacing** or **Distribute vertical spacing** (also in the top toolbar, next to align buttons).

- **Change text arrangement inside a text box** →  
  Select the text → In the right-hand panel (Properties), under **Text** section, adjust:
  - Alignment (left, center, right, justify)
  - Auto Width, Auto Height, or Fixed Size
  - Line height, paragraph spacing, etc.

- **Arrange order (forward/backward)** →  
  Select the text, right-click → **Arrange** →  
  - Bring Forward  
  - Bring to Front  
  - Send Backward  
  - Send to Back

---

**Quick Commands in Figma:**
- `Ctrl + Alt + G` (or `Cmd + Option + G` on Mac) → Group layers (for easier arrangement)
- `Shift + A` → Add Auto Layout (great for arranging text automatically)

---

**Summary:**  
- To **align and distribute** → Use top toolbar.  
- To **change z-order** (front/back) → Right-click → Arrange.  
- To **format inside** the text → Use right sidebar → Text settings.
