Here's an example of how you can start with Python:

### Step 1: Extract Text and Images

```python
import fitz  # PyMuPDF
import os
import pandas as pd

# Create a directory to store images
os.makedirs('extracted_images', exist_ok=True)

# Open the PDF
pdf_document = "The-DALL·E-2-prompt-book-v1.02.pdf"
pdf = fitz.open(pdf_document)

# Initialize lists to hold the data
data = []

# Iterate through pages
for page_num in range(len(pdf)):
    page = pdf.load_page(page_num)
    text = page.get_text("text")
    
    # Extract images
    images = page.get_images(full=True)
    for img_index, img in enumerate(images):
        xref = img[0]
        base_image = pdf.extract_image(xref)
        image_bytes = base_image["image"]
        image_ext = base_image["ext"]
        image_filename = f"extracted_images/page{page_num + 1}_img{img_index + 1}.{image_ext}"
        
        # Save image
        with open(image_filename, "wb") as img_file:
            img_file.write(image_bytes)
        
        # Append to data
        data.append({
            "page": page_num + 1,
            "text": text.strip(),
            "image_path": image_filename
        })

# Create a DataFrame
df = pd.DataFrame(data)
df.to_csv("dalle_prompt_book_dataset.csv", index=False)

print("Extraction complete. Dataset saved as 'dalle_prompt_book_dataset.csv'")
```

- Store the dataset in a CSV file with columns for page number, text content, and image paths.

```python
import pandas as pd
import tkinter as tk
from tkinter import ttk

# Load the dataset
dataset = pd.read_csv('final_image_prompt_dataset.csv')

# Function to update subcategories based on selected category
def update_subcategories(*args):
    selected_category = category_var.get()
    subcategories = dataset[dataset['Category'] == selected_category]['Subcategory'].unique()
    subcategory_menu['values'] = subcategories
    subcategory_menu.current(0)

# Function to generate the prompt based on selected category and subcategory
def generate_prompt():
    selected_category = category_var.get()
    selected_subcategory = subcategory_var.get()
    data = dataset[(dataset['Category'] == selected_category) & (dataset['Subcategory'] == selected_subcategory)]
    if not data.empty:

        examples = data['Examples'].values[0]
        prompt = f"Generate an image of {selected_subcategory} with details: {description}. Examples: {examples}."
        prompt_text.set(prompt)

# Create the main Tkinter window

root.title("Image Prompt Generator")

# Category selection
category_label = tk.Label(root, text="Select Category:")
category_label.grid(row=0, column=0, padx=10, pady=10)

category_var = tk.StringVar()
category_menu = ttk.Combobox(root, textvariable=category_var)
categories = dataset['Category'].unique()
category_menu['values'] = categories
category_menu.grid(row=0, column=1, padx=10, pady=10)
category_menu.bind('<<ComboboxSelected>>', update_subcategories)

# Subcategory selection
subcategory_label = tk.Label(root, text="Select Subcategory:")
subcategory_label.grid(row=1, column=0, padx=10, pady=10)

subcategory_var = tk.StringVar()

subcategory_menu.grid(row=1, column=1, padx=10, pady=10)

# Generate prompt button
generate_button = tk.Button(root, text="Generate Prompt", command=generate_prompt)
generate_button.grid(row=2, column=0, columnspan=2, padx=10, pady=10)

# Display prompt
prompt_text = tk.StringVar()
prompt_label = tk.Label(root, textvariable=prompt_text, wraplength=400)
prompt_label.grid(row=3, column=0, columnspan=2, padx=10, pady=10)
```

#### Overview of the Application
This guide will help you create a standalone Tkinter application that generates image prompts based on a dataset. The application allows you to select a category and subcategory, and then generates a detailed image prompt based on the selected data.

### Step 1: Install Required Software


### Step 3: Prepare Your Workspace

2. **Create a New Folder for Your Project**:
   - In VSCode, click on "File" > "Open Folder" and create/select a folder for your project.

   - In VSCode, click on "File" > "New File" and save it as `image_prompt_generator.py`.

### Step 4: Download the Dataset

1. **Download the Final Dataset**:
   - [Download the final image prompt dataset](sandbox:/mnt/data/final_image_prompt_dataset.csv).
   - Save the downloaded CSV file in your project folder.

### Step 5: Add the Code

1. **Copy and Paste the Following Code into `image_prompt_generator.py`**:

   ```python
   import pandas as pd
   import tkinter as tk
   from tkinter import ttk

   # Load the dataset
   dataset = pd.read_csv('final_image_prompt_dataset.csv')

   # Function to update subcategories based on selected category
   def update_subcategories(*args):
       selected_category = category_var.get()
       subcategories = dataset[dataset['Category'] == selected_category]['Subcategory'].unique()
       subcategory_menu['values'] = subcategories
       subcategory_menu.current(0)

   # Function to generate the prompt based on selected category and subcategory
   def generate_prompt():
       selected_category = category_var.get()
       selected_subcategory = subcategory_var.get()
       data = dataset[(dataset['Category'] == selected_category) & (dataset['Subcategory'] == selected_subcategory)]
       if not data.empty:
           description = data['Description'].values[0]
           examples = data['Examples'].values[0]
           prompt = f"Generate an image of {selected_subcategory} with details: {description}. Examples: {examples}."
           prompt_text.set(prompt)

   # Create the main Tkinter window
   root = tk.Tk()
   root.title("Image Prompt Generator")

   # Category selection
   category_label = tk.Label(root, text="Select Category:")
   category_label.grid(row=0, column=0, padx=10, pady=10)

   category_var = tk.StringVar()
   category_menu = ttk.Combobox(root, textvariable=category_var)
   categories = dataset['Category'].unique()
   category_menu['values'] = categories
   category_menu.grid(row=0, column=1, padx=10, pady=10)
   category_menu.bind('<<ComboboxSelected>>', update_subcategories)

   # Subcategory selection
   subcategory_label = tk.Label(root, text="Select Subcategory:")
   subcategory_label.grid(row=1, column=0, padx=10, pady=10)

   subcategory_var = tk.StringVar()
   subcategory_menu = ttk.Combobox(root, textvariable=subcategory_var)
   subcategory_menu.grid(row=1, column=1, padx=10, pady=10)

   # Generate prompt button
   generate_button = tk.Button(root, text="Generate Prompt", command=generate_prompt)
   generate_button.grid(row=2, column=0, columnspan=2, padx=10, pady=10)

   # Display prompt
   prompt_text = tk.StringVar()
   prompt_label = tk.Label(root, textvariable=prompt_text, wraplength=400)
   prompt_label.grid(row=3, column=0, columnspan=2, padx=10, pady=10)

   # Run the Tkinter main loop
   root.mainloop()
   ```

### Step 6: Run the Application


### Step 7: Use the Application

3. **Click "Generate Prompt"** to see the generated image prompt displayed.

To compile the Python Tkinter application into a standalone executable, you can use a tool like `PyInstaller`. This will allow you to distribute the application without requiring users to have Python installed. Here are the steps to do this:

### Step-by-Step Guide to Compile the Tkinter Application

1. **Install PyInstaller**:
   Open Command Prompt or Terminal and install PyInstaller using `pip`:
   ```bash
   pip install pyinstaller
   ```

2. **Prepare Your Python Script**:
   Ensure your script (`image_prompt_generator.py`) and the dataset (`final_image_prompt_dataset.csv`) are in the same directory.

3. **Compile the Script**:
   Use PyInstaller to compile the script into an executable:
   ```bash
   pyinstaller --onefile --add-data "final_image_prompt_dataset.csv;." image_prompt_generator.py
   ```
   - `--onefile`: This flag tells PyInstaller to bundle everything into a single executable.
   - `--add-data`: This flag is used to include additional files. The syntax is `source;destination`, where `source` is the path to the file and `destination` is the relative path where the file should be included in the bundle.
   
   For Windows, use a semicolon `;` to separate `source` and `destination`. For macOS/Linux, use a colon `:`.

4. **Locate the Executable**:
   After PyInstaller finishes, you will find the executable in the `dist` directory within your project folder.

### Example Commands

Here is the full command sequence you would use in your terminal:

