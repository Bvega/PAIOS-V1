Here is a list of descriptions for body languages that can be used for prompting characters:

1. **Confident Stance**
   - Description: Standing tall with shoulders back, chest out, and feet firmly planted. Arms are either at the sides or on the hips.
   
2. **Nervous Fidgeting**
   - Description: Shifting weight from one foot to the other, tapping fingers or feet, avoiding eye contact, and frequently adjusting clothing or accessories.
   
3. **Relaxed Posture**
   - Description: Leaning back in a chair with arms casually draped, legs slightly apart, and a slight smile on the face. 

4. **Aggressive Stance**
   - Description: Leaning forward with clenched fists, feet apart, and a glaring expression. Muscles are tense and ready for action.

5. **Defensive Position**
   - Description: Arms crossed over the chest, legs slightly bent, and body slightly turned away from the source of threat. Eyebrows furrowed.

6. **Joyful Movement**
   - Description: Bouncing on the toes, wide smile, eyes crinkled at the corners, arms open or raised, and body swaying with happiness.

7. **Thoughtful Pose**
   - Description: Hand on the chin, eyes slightly squinted, head tilted to one side, and body still. Sometimes pacing slowly while thinking.

8. **Fearful Expression**
   - Description: Wide eyes, mouth slightly open, body slightly crouched, and arms close to the body as if trying to protect oneself.

9. **Surprised Reaction**
   - Description: Eyes wide open, mouth agape, eyebrows raised, and hands either covering the mouth or spread out in shock.

10. **Angry Posture**
    - Description: Red face, clenched fists, tense muscles, and body leaning forward. Breathing is heavy and rapid.

11. **Sad Body Language**
    - Description: Shoulders slumped, head down, hands in pockets or covering the face, and slow, shuffling movements.

12. **Bored Attitude**
    - Description: Slouched posture, head resting on hand, eyes half-closed or looking away, and feet tapping out of impatience.

13. **Flirtatious Gesture**
    - Description: Playful smile, prolonged eye contact, light touching of hair or arm, and a slightly tilted head.

14. **Curious Exploration**
    - Description: Leaning forward slightly, eyes wide and focused, head tilted to one side, and hands reaching out or pointing.

15. **Suspicious Glance**
    - Description: Narrowed eyes, slightly turned head, tense jaw, and arms crossed or hands on hips.

16. **Excited Energy**
    - Description: Bouncing up and down, wide eyes, big smile, clapping hands, and rapid movements.

17. **Confused Expression**
    - Description: Furrowed brow, tilted head, pursed lips, and one hand scratching the head or chin.

18. **Proud Stance**
    - Description: Standing with chest out, hands on hips or behind the back, head held high, and a satisfied smile.

19. **Sleepy Posture**
    - Description: Drooping eyelids, yawning, slouched shoulders, and slow, unsteady movements.

20. **Determined Look**
    - Description: Firm jaw, narrowed eyes, forward-leaning stance, and fists clenched or arms rigid by the sides.

These descriptions can be used to create vivid and dynamic character prompts, helping to convey their emotions and attitudes through body language.

### 7. Example Code for a Web App
Here’s an example of how you might integrate this into a simple Flask web application:

```python
from flask import Flask, request, jsonify
import pandas as pd
import openai

app = Flask(__name__)

# Load your dataset
df = pd.read_csv("Fighting_Poses_and_Stances_Dataset.csv")

# Initialize OpenAI API
openai.api_key = 'your-api-key'

@app.route('/generate_prompt', methods=['POST'])
def generate_prompt():
    data = request.json
    index = data.get('index', 0)
    
    if index >= len(df):
        return jsonify({"error": "Index out of range"}), 400
    
    example = df.iloc[index]
    prompt = f"Describe a scene: {example['Prompt']}"
    
    response = openai.Completion.create(
        engine="davinci-codex",  # Use the appropriate engine
        prompt=prompt,
        max_tokens=150
    )
    
    return jsonify({"prompt": prompt, "response": response.choices[0].text.strip()})

if __name__ == '__main__':
    app.run(debug=True)
```

This code sets up a Flask server that takes a POST request with an index to generate a prompt based on the dataset and returns the generated text.
