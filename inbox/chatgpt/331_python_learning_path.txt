## **DAY 1: What is Python?**

### **Goal:**  
Understand what Python is, its history, and what it's used for.

### **Steps:**

1. **Read this article:**  
   [Python.org - About](https://www.python.org/about/)

2. **Watch (Optional):**  
   [Python Explained Simply ŌĆō FreeCodeCamp Intro](https://www.youtube.com/watch?v=rfscVS0vtbw)

3. **Key Concepts to Learn:**
   - Python is a high-level, interpreted language.
   - It emphasizes readability and simplicity.
   - Used in web dev, AI, data science, scripting, automation, etc.

4. **Quick Code Check:**
   Open any Python environment (like [Replit](https://replit.com/) or [Python Tutor](https://pythontutor.com/)) and run:
   ```python
   print("Hello, Python!")
   ```

5. **Task:**
   - Write 3 things you would like to build using Python.

---

## **DAY 2: Variables**

### **Goal:**  
Learn how to store and update data in Python.

### **Steps:**

1. **Key Concepts:**
   - Variables hold data.
   - Syntax: `variable_name = value`

2. **Examples:**
   ```python
   name = "Alice"
   age = 25
   height = 5.7
   is_student = True

   print(name, age, height, is_student)
   ```

3. **Try It Yourself:**
   - Create 3 variables: `your_name`, `your_age`, and `your_goal`
   - Print them in one sentence using `print()`.

4. **Explore More:**
   [W3Schools - Python Variables](https://www.w3schools.com/python/python_variables.asp)

---

## **DAY 3: Data Types**

### **Goal:**  
Understand basic Python data types.

### **Steps:**

1. **Common Types:**
   - `int`: whole numbers
   - `float`: decimal numbers
   - `str`: text
   - `bool`: `True` or `False`

2. **Examples:**
   ```python
   x = 10            # int
   pi = 3.14         # float
   name = "Alice"    # str
   is_active = True  # bool

   print(type(x), type(pi), type(name), type(is_active))
   ```

3. **Task:**
   - Create one variable of each type.
   - Use `type()` to confirm their type.

4. **Explore More:**
   [RealPython - Data Types](https://realpython.com/python-data-types/)

---

## **DAY 4: Basic Math**

### **Goal:**  
Use arithmetic operations in Python.

### **Steps:**

1. **Operators to Learn:**
   - `+`, `-`, `*`, `/`, `//`, `%`, `**`

2. **Examples:**
   ```python
   a = 8
   b = 3

   print(a + b)   # 11
   print(a - b)   # 5
   print(a * b)   # 24
   print(a / b)   # 2.666...
   print(a // b)  # 2
   print(a % b)   # 2
   print(a ** b)  # 512
   ```

3. **Mini Challenge:**
   - Calculate the area of a circle with radius 7.

   ```python
   radius = 7
   area = 3.14 * (radius ** 2)
   print("Area of circle:", area)
   ```

4. **Explore More:**
   [Python Math Operators](https://www.w3schools.com/python/python_operators.asp)

---

## **DAY 5: Operators (Comparison & Logic)**

### **Goal:**  
Use comparison and logical operators in Python.

### **Steps:**

1. **Comparison Operators:**
   - `==`, `!=`, `>`, `<`, `>=`, `<=`

2. **Logical Operators:**
   - `and`, `or`, `not`

3. **Examples:**
   ```python
   a = 5
   b = 10

   print(a == b)       # False
   print(a != b)       # True
   print(a > 3 and b < 20)  # True
   print(not a > b)    # True
   ```

4. **Mini Task:**
   Write 3 expressions using:
   - `and`
   - `or`
   - `not`

5. **Explore More:**
   [GeeksforGeeks - Operators in Python](https://www.geeksforgeeks.org/python-operators/)
