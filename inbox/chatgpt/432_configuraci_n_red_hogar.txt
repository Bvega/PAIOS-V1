### Paso 1: **Configura el Router**

1. **Accede al router:** desde navegador escribe la IP gateway: `192.168.254.1`
2. **Define la red local:**
   - Rango de IPs LAN: `192.168.254.2 – 192.168.254.254`
   - DHCP: Activado para dispositivos dinámicos (móviles, etc.)
   - Asignaciones fijas:
     - NAS: `192.168.254.64`
     - PC1: `192.168.254.74`
     - PC2: `192.168.254.73`
     - Impresora: (asignar IP fija, por ejemplo `192.168.254.50`)

3. **Wi-Fi:**
   - SSID principal: `WINS-6045746`
   - Clave segura WPA2
   - (Opcional) Red de invitados si quieres aislar dispositivos móviles.

4. **Firewall / Seguridad:**
   - Desactivar administración remota si no es necesario.
   - Cambiar clave de acceso al router.

---

### Paso 2: **Configura el NAS (Kyoyu)**

1. Conecta el NAS físicamente al router.
2. Accede mediante navegador a: `192.168.254.64`
3. Configura:
   - IP fija dentro del NAS.
   - Habilita carpetas compartidas.
   - Configura usuarios y permisos (por ejemplo: `admin`, `user`).
   - Habilita protocolos: **SMB**, **FTP**, **DLNA** para las TV.

4. Configura respaldo automático (opcional).

---

### Paso 3: **Configura el Switch**

El switch solo replica puertos, no necesita configuración avanzada si es no gestionado.
- Conecta todos los dispositivos cableados:
  - PC1, PC2, impresora, TV2, TV3.

(Opcional si es un switch gestionado: asigna VLAN para segmentar tráfico).

# Plan de acción completo para tu MediaSmart Server como NAS profesional (OpenMediaVault)

---

## Fase 1: Preparación

### 1.1. Herramientas necesarias

- 1 USB de al menos 8 GB
- Otro ordenador para preparar el USB
- Software:  
  - [Balena Etcher](https://www.balena.io/etcher/) o [Rufus](https://rufus.ie)
- **Imagen de OpenMediaVault**  
  [Descargar OMV](https://www.openmediavault.org/download.html)
  → Te recomiendo versión 6.x estable

---

### 1.2. Preparar USB de instalación

1. Inserta el USB en el PC.
2. Abre Rufus o Balena Etcher.
3. Selecciona la ISO de OpenMediaVault.
4. Configura:
   - **Tipo de partición:** MBR
   - **Sistema de archivos:** FAT32
   - **Modo de arranque:** BIOS (NO UEFI)

5. Graba la ISO.

---

## Fase 2: Hardware MediaSmart

### 2.1. Preparar el servidor
- Inserta un disco duro limpio en el MediaSmart Server.
- Conecta el USB de instalación.
- Conecta el servidor al router con cable Ethernet.
- Conecta el servidor a la corriente eléctrica.

### 2.2. Instalación "Headless"
Como tu MediaSmart no tiene salida VGA nativa, vamos a hacerlo **sin monitor**, por red.

El truco:  
OpenMediaVault se instala en modo automático si no detecta teclado/monitor.  
Luego accedes por IP en tu red.

---

## Fase 3: Instalación y acceso inicial

### 3.1. Encender el servidor

- Enciende tu servidor con el USB insertado.
- Espera ~10-15 minutos (la instalación se completa automáticamente en la mayoría de estos equipos).
- Verifica tu red para encontrar la IP del servidor:
  - Usa el router, aplicación de escaneo de red como **Fing** o **Advanced IP Scanner**.
  - El nombre de host predeterminado será algo como: `openmediavault.local`

### 3.2. Accede desde tu PC:
- Navegador: `http://[IP-DEL-SERVIDOR]/`
- Usuario: `admin`
- Contraseña: `openmediavault`

---

## Fase 4: Configuración básica

1. **Cambia la contraseña de administrador.**
2. **Asigna IP fija:**
   - Por ejemplo: `192.168.254.70` (para que encaje en tu red actual).
3. **Actualiza el sistema** en la pestaña de actualización de OMV.
4. **Configura notificaciones por email** (opcional pero recomendado).
5. **Crea usuarios** para acceso a las carpetas compartidas.

---

## Fase 5: Configuración de discos

1. **Identifica el disco instalado.**
2. **Formatea y crea sistema de archivos** (EXT4 recomendado).
3. **Crea un volumen de almacenamiento.**
4. **Habilita comparticiones SMB/CIFS** para que las veas desde Windows/Mac.

---

## Fase 6: Servicios opcionales recomendados

### Opcional pero muy recomendable:
- **Plex Media Server:** para servir videos a tus TVs.
- **DLNA:** para que las TVs detecten automáticamente películas/música.
- **Respaldo automático de PC:** configurar rsync o plugin de copia automática.
- **Nextcloud:** para crear tu nube personal.
- **Docker:** para contenedores y ampliar funcionalidades fácilmente.

---

## Fase 7: Integración con tu red existente

✅ Asegurarte que tu nuevo servidor:
- Se comunique con tu NAS Kyoyu.
- Sea visible para tus PCs (PC1, PC2).
- Que las TVs accedan al contenido multimedia.
- Que las copias de seguridad automáticas funcionen.
