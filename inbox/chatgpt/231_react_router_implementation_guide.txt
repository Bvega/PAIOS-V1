ASSISTANT:
• React Router enables seamless navigation without full page reloads, improving speed and user flow in SPAs.  
• Challenges: handling non-existent routes, deep linking, or async data loading. Solve with `Route path="*"` for 404s, `useParams`, and lazy loading.  
• Use Context API to store auth state; wrap protected routes in a component that checks auth and redirects using `<Navigate />` if unauthorized.

ASSISTANT:
• **Which of the advanced React concepts covered in this module do you find most useful and why?**  
Hooks are the most useful—especially `useEffect` and custom hooks—because they allow clean, modular logic and lifecycle control without class components.

• **How do you see the combination of Hooks, Context API, and React Router benefiting your future projects?**  
Together, they enable scalable apps with centralized state (Context), modular logic (Hooks), and seamless navigation (Router), streamlining development and improving UX.

• **What challenges did you face while working with these concepts, and how did you overcome them?**  
Challenge: managing prop drilling and side effects. Solution: used Context to avoid deep prop chains and `useEffect` with clear dependencies to control re-renders.
