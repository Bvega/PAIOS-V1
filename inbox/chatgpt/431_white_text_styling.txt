## Step 1: Understand the Concept

Right now you have a **background-color** on the `<body>` and `<section>`:
```css
body {
  background-color: rgb(30, 255, 244);
}

section {
  background-color: rgba(187, 39, 245, 0.445);
}
```

You want to **replace** that with a **background image**.

✅ **Concept:**  
In CSS, you use:
```css
background-image: url('your-image-path.jpg');
```

You can also control:
- **Position**: where the image sits (e.g., center, top, left)
- **Size**: cover the whole background, or repeat
- **Repeat**: repeat the image or no
- **Attachment**: fixed or scrolls with page

Our goal is:  
✔️ Full page background image  
✔️ Scalable (covers the screen)

---

## Step 2: Clean Code Step-by-Step

### 2.1 — Remove the `background-color` from `<body>` and `<section>`  
(We are replacing it)

### 2.2 — Add background-image to `<body>` (this affects the whole page)

### 2.3 — Optional: Add background-image to `<section>` if you want a separate image inside sections (I'll show both)

---

## Step 3: Apply in Your CSS

Here’s your **new `styles.css`** ready to copy-paste:

```css
section {
    display: flex;
    text-align: center;
    align-items: center;
    justify-content: space-around;
    border-style: solid;
    background: rgba(187, 39, 245, 0.445); /* Optional overlay color */
    padding: 20px;
}

body {
    text-align: center;
    background-image: url('https://images.unsplash.com/photo-1522199755839-a2bacb67c546?ixlib=rb-4.0.3&auto=format&fit=crop&w=1950&q=80'); /* Example background */

}

    margin-bottom: 20px;
}

img {
    max-height: 250px;
    margin-top: 20px;
}
```

### ✅ Step 1: Concept

Your text is using the default browser color (black).  
To change all letters to **white**, we apply:

```css
color: white;
```

We apply this to:
- `body { color: white; }`  
This will make all text inside the body white, including headings, lists, paragraphs, etc.

If you want, we can also style specific elements for better control later.

---

### ✅ Step 2: Apply in CSS

Here is your **updated `styles.css`**, ready to copy and paste:

```css
section {
    display: flex;
    text-align: center;
    align-items: center;
    justify-content: space-around;
    border-style: solid;
    background: rgba(187, 39, 245, 0.445); /* Optional overlay color */
    padding: 20px;
}

body {
    text-align: center;
    background-image: url('https://images.unsplash.com/photo-1522199755839-a2bacb67c546?ixlib=rb-4.0.3&auto=format&fit=crop&w=1950&q=80');
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat;
    background-attachment: fixed;
    color: white; /* ✅ This makes all text white */
}

ul {
    display: inline;
    text-align: center;
    align-items: center;
    margin-right: 30px;
}

.facts {
    display: flex;
    margin-bottom: 20px;
}

img {
    max-height: 250px;
    margin-top: 20px;
}
```
