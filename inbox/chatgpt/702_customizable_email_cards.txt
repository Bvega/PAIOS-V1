### Step-by-Step Instructions to Personalize and Send Emails Using Outlook with Mail Merge

Below are detailed steps to set up and send personalized emails in Outlook using a Mail Merge with an HTML email template.

---

#### **Step 1: Prepare Your Email Recipients List**
1. **Open Microsoft Excel**:
   - Create a new spreadsheet.
   - Add the following columns:
     - **First Name** (e.g., John, Sarah, etc.)
     - **Email Address** (e.g., john.doe@example.com)
     - Any additional columns you may need (e.g., Last Name, Greeting).

2. **Fill in the Data**:
   - Add the recipient names and email addresses to the rows under the appropriate columns.

3. **Save the File**:
   - Save the spreadsheet as an Excel Workbook (*.xlsx) with a clear name, such as `Email_Recipients.xlsx`.

---

#### **Step 2: Create Your HTML Email Template**
1. **Copy the HTML Code**:
   - Use the HTML template provided above (with `{{recipient_name}}` as the placeholder for the recipient's name).

2. **Save the HTML File**:
   - Paste the HTML code into a plain text editor like Notepad.
   - Save the file with a `.html` extension, e.g., `Holiday_Email_Template.html`.

3. **Preview the Email**:
   - Open the `.html` file in a web browser to check the layout and appearance.

---

#### **Step 3: Set Up the Mail Merge in Word**
1. **Open Microsoft Word**:
   - Go to **Mailings** in the top menu.
   - Click on **Start Mail Merge** > **Email Messages**.

2. **Connect the Recipient List**:
   - Click on **Select Recipients** > **Use an Existing List**.
   - Browse and select the Excel file (`Email_Recipients.xlsx`) you created earlier.
   - In the dialog box, choose the correct worksheet (usually `Sheet1$`) and click **OK**.

3. **Insert Personalized Fields**:
   - In the Word document, click **Insert Merge Field** > Select the field you want to personalize (e.g., `First Name`).
   - Replace `{{recipient_name}}` in your HTML template with the inserted merge field:
     - Example: Replace `{{recipient_name}}` with `«First_Name»`.

4. **Insert the HTML Template**:
   - Press **Ctrl + A** to select all content in the `.html` file (open it in a text editor).
   - Paste it into the Word document.

5. **Preview the Emails**:
   - Click **Preview Results** to see how the email looks for each recipient.

---

#### **Step 4: Send the Personalized Emails via Outlook**
1. **Set the Email Subject**:
   - Click **Finish & Merge** > **Send Email Messages**.
   - In the dialog box:
     - Set **To** to the column containing email addresses (e.g., `Email Address`).
     - Set **Subject line** to your desired subject (e.g., "Warm Holiday Wishes!").
     - Set **Mail format** to **HTML**.
   - Click **OK** to send the emails.

2. **Verify Sent Emails**:
   - Open Outlook and go to the **Sent Items** folder to ensure all emails were sent.

---

#### **Step 5: Test Before Sending to All Recipients**
1. **Send a Test Email**:
   - Create a test entry in your Excel file with your own email address.
   - Follow the above steps to send a single test email.
   - Review the email for formatting and personalization.

2. **Fix Any Issues**:
   - If the email doesn’t display correctly, adjust the HTML template or Excel file and repeat the test.

---

#### Additional Notes:
- Ensure that Outlook is set as your default email application.
- If you have a large recipient list, consider sending emails in smaller batches to avoid email server limitations.
- You can add more personalization fields (e.g., company name, custom messages) by adding additional merge fields to your Excel file and HTML template.
