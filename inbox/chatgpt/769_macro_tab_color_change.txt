ASSISTANT:
Yes, you can modify the macro to remove `TintAndShade` and use only the pure accent colors. This way, the color applied to the tabs will be exactly the specified theme color without any modification. Here is your macro adjusted accordingly:

```vba
Sub TabColorChange()
'
' TabColorChange Macro
'

    ' Set pure accent colors for each tab without TintAndShade
    Sheets("Summary").Select
    With ActiveWorkbook.Sheets("Summary").Tab
        .ThemeColor = xlThemeColorAccent3
    End With

    Sheets("Submitted proposal").Select
    With ActiveWorkbook.Sheets("Submitted proposal").Tab
        .ThemeColor = xlThemeColorAccent4
    End With

    Sheets("Meeting Rooms").Select
    With ActiveWorkbook.Sheets("Meeting Rooms").Tab
        .ThemeColor = xlThemeColorAccent5
    End With

    Sheets("Turned down").Select
    With ActiveWorkbook.Sheets("Turned down").Tab
        .ThemeColor = xlThemeColorAccent6
    End With

End Sub
```

In this modified version:
- The line `.TintAndShade = 0.799981688894314` has been removed for each tab setting.
- Now, the pure theme color (`xlThemeColorAccentX`) will be used for each sheet tab.

This should give you the exact theme accent color without any tinting or shading.
