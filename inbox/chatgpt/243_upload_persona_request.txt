You’ve been tasked with developing two common custom Hooks: usePagination for managing client-side pagination logic, and useDebounce for debouncing rapidly changing values, often used with search inputs.

This lab will challenge you to think about hook design, state management within hooks, handling dependencies, and ensuring your hooks are generic enough for reuse.

Learning Objectives
Upon successful completion of this lab, you will be able to:

Design and implement reusable custom React Hooks from scratch.

Apply custom Hooks to solve common problems like pagination and debouncing input.
Consider basic performance aspects (e.g., memoization) where applicable within custom Hooks.

useDebounce Hook
Objective: Create a hook to debounce a rapidly changing value.

Functionality:

**Implementing `usePagination` Hook**

We'll build the hook step-by-step:

1. **State Management**  
   - Use `useState` to store the current page.
2. **Derived Values**  
   - Compute `totalPages` with `Math.ceil`.
   - Calculate `startIndex` and `endIndex` based on `currentPage` and `itemsPerPage`.
   - Determine `itemsOnCurrentPage`, handling the last page edge case.
3. **Navigation Functions**  
   - `setPage` clamps the page number within valid bounds.  
   - `nextPage` and `prevPage` advance or rewind if permitted.
4. **Booleans for UI**  
   - `canNextPage` and `canPrevPage` indicate if navigation is possible.
5. **Effect for Bounds Safety**  
   - If `totalItems` or `itemsPerPage` change, ensure `currentPage` remains ≤ `totalPages`.

### Full code for `src/hooks/usePagination.ts`

```ts
// src/hooks/usePagination.ts
import { useState, useEffect, useCallback } from 'react';

interface UsePaginationResult {
  currentPage: number;
  totalPages: number;
  startIndex: number;
  endIndex: number;
  itemsOnCurrentPage: number;
  setPage: (page: number) => void;
  nextPage: () => void;
  prevPage: () => void;
  canNextPage: boolean;
  canPrevPage: boolean;
}

/**
 * Custom hook to manage pagination logic.
 *
 * @param totalItems - total number of items to paginate
 * @param itemsPerPage - number of items per page (default 10)
 * @param initialPage - starting page (default 1)
 * @returns pagination state and navigation functions
 */
export function usePagination(
  totalItems: number,
  itemsPerPage = 10,
  initialPage = 1
): UsePaginationResult {
  // State for current active page
  const [currentPage, setCurrentPage] = useState(initialPage);

  // Calculate total pages based on items and size
  const totalPages = Math.max(1, Math.ceil(totalItems / itemsPerPage));

  // Ensure currentPage stays within [1, totalPages]
  useEffect(() => {
    if (currentPage > totalPages) {
      setCurrentPage(totalPages);
    } else if (currentPage < 1) {
      setCurrentPage(1);
    }
  }, [currentPage, totalPages]);

  // Derive starting and ending indices for slicing
  const startIndex = (currentPage - 1) * itemsPerPage;
  // endIndex capped at last item
  const endIndex = Math.min(totalItems - 1, startIndex + itemsPerPage - 1);

  // Number of items actually on the current page
  const itemsOnCurrentPage = totalItems === 0 ? 0 : endIndex - startIndex + 1;

  // Clamp and set a specific page
  const setPage = useCallback(
    (page: number) => {
      const newPage = Math.min(Math.max(1, page), totalPages);
      setCurrentPage(newPage);
    },
    [totalPages]
  );

  // Advance to next page if possible
  const nextPage = useCallback(() => {
    if (currentPage < totalPages) {
      setCurrentPage((prev) => prev + 1);
    }
  }, [currentPage, totalPages]);

  // Go back to previous page if possible
  const prevPage = useCallback(() => {
    if (currentPage > 1) {
      setCurrentPage((prev) => prev - 1);
    }
  }, [currentPage]);

  // Booleans for UI button disabling
  const canNextPage = currentPage < totalPages;
  const canPrevPage = currentPage > 1;

  return {
    currentPage,
    totalPages,
    startIndex,
    endIndex,
    itemsOnCurrentPage,
    setPage,
    nextPage,
    prevPage,
    canNextPage,
    canPrevPage,
  };
}
```
