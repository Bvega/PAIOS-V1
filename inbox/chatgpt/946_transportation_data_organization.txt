To set up a web scraping tool to gather information from transportation companies' websites, we can use Python with libraries like `BeautifulSoup` and `requests`. Here's a step-by-step guide to building this scraper:

### Prerequisites:
1. **Install Python** (if not already installed) from [python.org](https://www.python.org/downloads/).
2. **Install Required Libraries**:
   - `requests`: To send HTTP requests to websites.
   - `BeautifulSoup4`: To parse and extract data from the HTML content.

   Install them using pip:
   ```bash
   pip install requests beautifulsoup4 openpyxl
   ```

### Steps to Create the Web Scraping Script:

1. **Set Up the Script**:
   - We'll make HTTP requests to search for transportation company data.
   - Parse the HTML content of the pages.
   - Extract the relevant information (phone numbers, addresses, etc.).
   - Update the Excel sheet with the gathered data.

2. **Code for Web Scraping and Updating Spreadsheet**:

   Here's a basic Python script to get started:

```python
import requests
from bs4 import BeautifulSoup
import pandas as pd

# Function to fetch the website content
def get_website_content(url):
    try:
        response = requests.get(url)
        if response.status_code == 200:
            return response.text
        else:
            return None
    except Exception as e:
        print(f"Error fetching {url}: {e}")
        return None

# Function to scrape phone number, address, email from website content
def scrape_company_info(content):
    soup = BeautifulSoup(content, 'html.parser')
    phone = address = email = None
    
    # Example of how to find phone, address, and email (this can vary by site)
    phone = soup.find('a', {'href': lambda href: href and 'tel:' in href})
    email = soup.find('a', {'href': lambda href: href and 'mailto:' in href})
    address = soup.find('address')  # This is a common tag for addresses

    return {
        'phone': phone.get_text() if phone else '',
        'email': email.get_text() if email else '',
        'address': address.get_text() if address else ''
    }

# Load the Excel sheet
file_path = '/mnt/data/Transportation_Companies.xlsx'
df = pd.read_excel(file_path)

# Loop over each company in the spreadsheet
for index, row in df.iterrows():
    company_name = row['Company Name']
    search_url = f"https://www.google.com/search?q={company_name.replace(' ', '+')}+official+website"
    
    # Fetch search results (this part might need manual refinement, since scraping Google directly is not ideal)
    search_results = get_website_content(search_url)
    
    if search_results:
        company_info = scrape_company_info(search_results)
        
        # Update the dataframe
        df.at[index, 'Phone Number'] = company_info['phone']
        df.at[index, 'Email'] = company_info['email']
        df.at[index, 'Full Address'] = company_info['address']

# Save the updated Excel file
updated_file_path = '/mnt/data/Updated_Transportation_Companies.xlsx'
df.to_excel(updated_file_path, index=False)

print(f"Updated spreadsheet saved to {updated_file_path}")
```

### Explanation:
1. **Requests and BeautifulSoup**: We use `requests` to send HTTP requests to websites and `BeautifulSoup` to parse and extract the required information from the HTML content.
2. **Scraping Logic**: The script looks for common HTML elements where contact info (like phone numbers, emails, and addresses) might be located.
3. **Updating Excel**: The results are added back to the Excel file and saved as a new file.

### Important Notes:
- **Ethical Scraping**: Make sure to follow the site's `robots.txt` rules to avoid violating their terms of service.
- **Google Search**: This example assumes searching for company websites using Google. Direct scraping of Google results is not recommended; you should either use an API like the Google Custom Search API or manually input company websites.

### Further Enhancement:
1. **Use an API**: Use services like the Google Custom Search API to automate gathering websites and other info more reliably.
2. **Dynamic Websites**: If a company’s website uses JavaScript to load data, you'll need a tool like `Selenium` to handle those cases.

ASSISTANT:
Here’s a step-by-step guide to create a simple web scraping app with a **Tkinter** interface in Python. This app will allow you to search for transportation company websites and scrape the information like phone numbers, addresses, and emails.

### Prerequisites:
1. **Install Python** from [python.org](https://www.python.org/downloads/).
2. Install the necessary libraries using pip:
   ```bash
   pip install requests beautifulsoup4 openpyxl tkinter
   ```

### The App Code:

```python
import requests
from bs4 import BeautifulSoup
import pandas as pd
import tkinter as tk
from tkinter import filedialog, messagebox
import openpyxl

# Function to get content from a website
def get_website_content(url):
    try:
        response = requests.get(url)
        if response.status_code == 200:
            return response.text
        else:
            return None
    except Exception as e:
        print(f"Error fetching {url}: {e}")
        return None

# Function to scrape company details
def scrape_company_info(content):
    soup = BeautifulSoup(content, 'html.parser')
    phone = address = email = None
    
    phone = soup.find('a', {'href': lambda href: href and 'tel:' in href})
    email = soup.find('a', {'href': lambda href: href and 'mailto:' in href})
    address = soup.find('address')  # This is a common tag for addresses
    
    return {
        'phone': phone.get_text() if phone else '',
        'email': email.get_text() if email else '',
        'address': address.get_text() if address else ''
    }

# Function to start the scraping and update Excel
def start_scraping(file_path):
    try:
        df = pd.read_excel(file_path)
        
        for index, row in df.iterrows():
            company_name = row['Company Name']
            search_url = f"https://www.google.com/search?q={company_name.replace(' ', '+')}+official+website"
            
            search_results = get_website_content(search_url)
            if search_results:
                company_info = scrape_company_info(search_results)
                
                df.at[index, 'Phone Number'] = company_info['phone']
                df.at[index, 'Email'] = company_info['email']
                df.at[index, 'Full Address'] = company_info['address']
        
        # Save the updated Excel file
        updated_file_path = filedialog.asksaveasfilename(defaultextension=".xlsx", filetypes=[("Excel files", "*.xlsx")])
        df.to_excel(updated_file_path, index=False)
        messagebox.showinfo("Success", f"Updated spreadsheet saved to {updated_file_path}")
    
    except Exception as e:
        messagebox.showerror("Error", f"An error occurred: {e}")

# GUI setup
def open_file():
    file_path = filedialog.askopenfilename(filetypes=[("Excel files", "*.xlsx")])
    if file_path:
        start_scraping(file_path)

# Main window
root = tk.Tk()
root.title("Transportation Companies Scraper")

label = tk.Label(root, text="Click the button to select the Excel file")
label.pack(pady=10)

button = tk.Button(root, text="Select Excel File", command=open_file)
button.pack(pady=10)

root.geometry("400x200")
root.mainloop()
```

### Explanation of the Code:

1. **Tkinter Interface**:
   - The app has a simple interface with a button to select an Excel file. Once selected, the app will start scraping.
   - It allows you to save the updated Excel file after scraping, using a file dialog.
   
2. **Web Scraping Functions**:
   - `get_website_content(url)`: Sends an HTTP request to get the HTML content of the page.
   - `scrape_company_info(content)`: Extracts phone numbers, addresses, and emails from the HTML content.

3. **Excel Handling**:
   - Reads the company names from the provided Excel file.
   - Updates the file with the scraped data and saves it to a new location.

### Steps to Run the App:

1. **Save the Python Script**: Copy the above code into a `.py` file (e.g., `scraper_app.py`).
   
2. **Run the Script**: 
   ```bash
   python scraper_app.py
   ```
   This will launch the Tkinter GUI.

3. **Usage**:
   - Click the button to select your Excel file with company names.
   - The script will scrape for the companies' websites and extract information like phone numbers, addresses, and emails.
   - You’ll be prompted to save the updated Excel file after scraping.

### Limitations:
