## JavaScript Basics Tutorial for Beginners  
**Persona**: Software Engineer Teacher  
**Mood**: Friendly, Encouraging  

### 1. Introduction & Setup  
1. Ensure you have a modern browser console (Chrome, Firefox, Edge).  
2. Open your browser’s DevTools (usually F12 or right-click → Inspect → Console).  
3. For each exercise, you’ll type the code in the console and press **Enter** to run.  

---

### 2. F. Biggie Smalls  
**Goal**: Practice a basic `if…else` check on a number.  
```js
// 1. Create a number variable
let num = 42;

// 2. Use an if...else statement
if (num < 100) {
  console.log("little number");
} else {
  console.log("big number");
}
// Expected output: "little number"
```
- **Explanation**:  
  - `if (num < 100)`: checks if `num` is smaller than 100.  
  - `console.log("little number")`: runs if the condition is true.  
  - `else`: runs when the condition is false.

---

### 3. G. Monkey in the Middle  
**Goal**: Chain conditions with `if…else if…else`.  
```js
let value = 7;

if (value < 5) {
  console.log("little number");
} else if (value > 10) {
  console.log("big number");
} else {
  console.log("monkey");
}
// Expected output: "monkey"
```
- **Flow**:  
  1. Check `< 5` → no  
  2. Check `> 10` → no  
  3. Else → `"monkey"`

```js
function isVowel(char) {
  return "aeiouAEIOU".includes(char);
```

```js
function getTwoLengths(str1, str2) {
```

#### F. `getMultipleLengths`
```js
function getMultipleLengths(arr) {
  return arr.map(s => s.length);
}
console.log(getMultipleLengths(["hello","what","is","up","dude"])); // [5,4,2,2,4]
```

#### G. `maxOfThree`
```js
function maxOfThree(a, b, c) {
  if(a >= b && a >= c) return a;
  else if(b >= a && b >= c) return b;
  else return c;
}
console.log(maxOfThree(6, 9, 1)); // 9
```

#### H. `printLongestWord`
```js
function printLongestWord(words) {
  let longest = words[0];
  for (let w of words) {
    if (w.length > longest.length) longest = w;
  }
  return longest;
}
console.log(printLongestWord(["BoJack","Princess","Diane","a","Max","Peanutbutter","big","Todd"]));
// "Peanutbutter"
```

```js
const c = 57;
const d = 16;
const e = "Kevin";

console.log(a < b);                     // 4 < 53 → true
console.log(c > d);                     // 57 > 16 → true
console.log("Name" === "Name");         // identical strings
// For the next two use only && or ||
console.log(true || false);             // true if either is true
console.log(false && false && false && false && false || true);
//          all false ANDs        OR true → true
console.log(false == false);            // equality of booleans
console.log(e === "Kevin");             // same string
// Chain comparisons using arithmetic
console.log(a + b === c);               // 4 + 53 = 57
console.log(a * d === a + d);           // 4*16 = 64, 4+16 = 20 → false?  
// Actually the prompt expects a simple arithmetic that makes true:
console.log(a * d / d === a);          // (4*16)/16 = 4
console.log(48 == "48");                // loose equality coerces string to number
```

# II. Loops

> **Reminder**: always use `let` for your loop counter.

### A. The Basics  
1. **0 to 10**  
   ```js
   for (let i = 0; i <= 10; i++) {
     console.log(i);
   }
   ```
2. **10 to 400**  
   ```js
   for (let i = 10; i <= 400; i++) {
     console.log(i);
   }
   ```
3. **Every 3rd number from 12 to 4000**  
   ```js
   for (let i = 12; i <= 4000; i += 3) {
     console.log(i);
   }
   ```

---

### B. Get Even  
Print 1–100, tagging evens:
```js
for (let i = 1; i <= 100; i++) {
  if (i % 2 === 0) {
    console.log(i + " is an even number");
  } else {
    console.log(i);
  }
}
```

### C. Give Me Five  
1. **High Five** for multiples of 5  
2. **Three is a crowd** for multiples of 3  
3. **Both** messages if divisible by 3 & 5  

```js
for (let i = 0; i <= 100; i++) {
  if (i % 3 === 0) {
    console.log(`I found a ${i}. Three is a crowd`);
  }
  if (i % 5 === 0) {
    console.log(`I found a ${i}. High five!`);
  }
}
```

### D. Savings Account  
1. **Sum** 1–10 → should be 55

   ```js
   let bank_account = 0;
   ```

# III. Arrays & Control Flow

### A. Talk About It  
1. Elements in an array are called _elements_ (or _items_).  
2. Arrays **do** maintain order.  
3. You could model a to-do list, playlist, or shopping cart.

---

### B. Easy Does It  
```js
const quotes = [
  "To be or not to be",
  "I think, therefore I am",
  "The only constant is change"
];
```

---

### C. Accessing Elements  
```js
const randomThings = [1, 10, "Hello", true];
// 1st element:
console.log(randomThings[0]);  // 1
// Change "Hello" to "World":
randomThings[2] = "World";
console.log(randomThings);     // [1,10,"World",true]
```

---

### D. Change Values  
```js
const ourClass = ["Salty", "Zoom", "Sardine", "Slack", "Github"];
// 3rd element:
console.log(ourClass[2]);  // "Sardine"
// Change "Github" → "Octocat":
ourClass[4] = "Octocat";
// Add "Cloud City":
ourClass.push("Cloud City");
console.log(ourClass);
```

---

### E. Mix It Up  
```js
const myArray = [5, 10, 500, 20];
// Add "Aegon" & another string:
myArray.push("Aegon", "Dragon");
// Remove 5 from start:
myArray.shift();
// Add "Bob Marley" to start:
myArray.unshift("Bob Marley");
// Remove last element:
myArray.pop();
// Reverse array:
myArray.reverse();

// Final state and method behavior:
console.log(myArray);
// .reverse() mutates the array in-place and also returns the reversed array.
```
