
**Common pitfalls that trigger your exact 404:**
- Frontend calls `/api/projects/` (trailing slash) while backend mounted deeper like `app.use('/api/projects/', ...)` plus route mismatch — usually Express is fine with slashes, but it’s best to be consistent and **omit** trailing slash on the client.
- Frontend calls `GET` when the server only has `POST` (or vice‑versa).
- Route never mounted in `server.js` (most common).
- Wrong base URL (e.g., pointing to `http://localhost:3000` or missing port).
- Vite proxy misconfig. If you use a proxy, ensure:
  ```js
  // vite.config.js
  export default {
    server: {
      proxy: {
        '/api': 'http://localhost:5000',
      },
    },
  };
  ```
