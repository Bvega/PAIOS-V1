```typescript
// Interface for Customer

  id: ID;
  name: string;
  email: string;
  loyaltyMember?: boolean; // optional
}

  id: 101,
  name: "Alice Smith",
  email: "alice@example.com",
  loyaltyMember: true,
};

```

### 🧠 Critical Thinking: Type Aliases vs. Interfaces

- **Use Interfaces When:**
  - You want to define the shape of an object.
  - You plan to use `extends` or implement classes.
  - The structure may be extended by others (interfaces are open and can be merged).

- **Use Type Aliases When:**
  - You are defining a union or intersection of types.
  - You need to define a primitive, tuple, or union (like `type ID = number | string`).
  - You don’t need extensibility, and prefer simplicity for aliasing complex types.

**Conclusion:** Interfaces are ideal for defining structured objects, especially when extensibility is a factor. Type aliases are great for simpler, flexible type definitions such as unions or function signatures.

### ### **Step 1: Define a Type Alias for ID**

A **type alias** is a way to name a custom type.

```ts
// This means ID can be either a number or a string
type ID = number | string;
```

🧠 **Why use a type alias?**  
It gives your code better readability and reuse. Instead of repeating `number | string`, you can just write `ID`.

---

### ### **Step 2: Create the `Customer` Interface**

An **interface** describes the shape of an object. Here's how to define it:

```ts
interface Customer {
  id: ID;               // uses the type alias we just defined
  name: string;         // customer’s name must be a string
  email: string;        // customer’s email must be a string
  loyaltyMember?: boolean;  // optional: true or false (or not included)
}
```

🧠 **What's that `?` after `loyaltyMember`?**  
It means this property is **optional**—the object may or may not have it.

---

### ### **Step 3: Write a Function to Use the Interface**

We now write a function that **takes a Customer** and returns a welcome message.

```ts
function createCustomer(customer: Customer): string {
  if (customer.loyaltyMember) {
    return `Welcome, ${customer.name}! Thanks for being a loyal customer!`;
  } else {
    return `Welcome, ${customer.name}!`;
  }
}
```

🧠 **What’s going on here?**
- We check if `loyaltyMember` is `true`.
- We return a message with the customer's name.

---

### ### **Step 4: Create a Customer Object and Call the Function**

Now let's make a sample customer and call our function:

```ts
const customer1: Customer = {
  id: 101,

  email: "alice@example.com",
  loyaltyMember: true,
```

```ts
// Step 1: Define a type alias 'ID' that can be either a number or a string
type ID = number | string;

// Step 2: Define the shape of a Customer object using an interface
interface Customer {
  id: ID;                    // The customer's unique ID (number or string)
  name: string;              // The customer's name
  email: string;             // The customer's email address
  loyaltyMember?: boolean;   // Optional: whether the customer is a loyalty program member
}

// Step 3: Function that returns a welcome message for the given customer
function createCustomer(customer: Customer): string {
  if (customer.loyaltyMember) {
    return `Welcome, ${customer.name}! Thanks for being a loyal customer!`;
  } else {
    return `Welcome, ${customer.name}!`;
  }
}

// Step 4: Wait until the page is fully loaded before attaching event listeners
window.onload = () => {
  // Get the form element and the message display element
  const form = document.getElementById("customerForm") as HTMLFormElement;
  const message = document.getElementById("message") as HTMLParagraphElement;

  // Step 5: When the form is submitted...
  form.onsubmit = (event) => {
    event.preventDefault(); // Prevent the form from reloading the page

    // Step 6: Get the input values from the form
    const idInput = (document.getElementById("id") as HTMLInputElement).value;
    const name = (document.getElementById("name") as HTMLInputElement).value;
    const email = (document.getElementById("email") as HTMLInputElement).value;

    // Step 7: Convert the form data into a Customer object
    const customer: Customer = {
      id: isNaN(Number(idInput)) ? idInput : Number(idInput), // Convert to number if possible
      name,
      email,
      loyaltyMember,
    };

    // Step 8: Display the welcome message
    message.textContent = createCustomer(customer);
  };
```
