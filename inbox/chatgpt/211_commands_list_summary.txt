Apply this middleware only to a new /api route path. Create a public route like / that does not require the key.

```js
// server.js  (place at project root)
import express from 'express';              // ESM syntax—Node ≥18 supports it natively
const app = express();
const PORT = process.env.PORT || 3000;

/* ---------- Activity 1: Global Request Logger ---------- */
app.use((req, _res, next) => {
  const now = new Date().toISOString();     // 2025-07-08T14:23:30.123Z
  console.log(`[${now}] ${req.method} ${req.path}`);
  next();                                   // hand control to next middleware/route
});

/* ---------- Simple public routes for testing ---------- */
app.get('/', (_req, res) => res.send('Hello from /'));
app.get('/test', (_req, res) => res.send('Hello from /test'));

/* ---------- Start server ---------- */
app.listen(PORT, () => console.log(`🚀 Server ready at http://localhost:${PORT}`));
```

```bash
# 4) Run the server
node server.js
# 5) In another terminal/tab, curl or hit with browser:

curl http://localhost:3000/test
```

```
[2025-07-08T14:25:04.789Z] GET /test
```
