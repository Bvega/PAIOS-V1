## 🧩 **Step 4: Implement the Product Base Class**

### 🔧 Goal
Create a base `Product` class to define shared properties and behavior for all products.

### 📄 File: `src/models/Product.ts`

Here’s the code with **clear comments and beginner-friendly explanations**:

```ts
// src/models/Product.ts

// The Product class is an abstract blueprint for all products in our inventory system.
// It defines common properties and methods shared by all product types.

export abstract class Product {
  constructor(
    public sku: string,   // Unique identifier for the product
    public name: string,  // Human-readable name of the product
    public price: number  // Base price of the product before tax
  ) {}

  // displayDetails returns a string with formatted product details
  displayDetails(): string {
    return `SKU: ${this.sku}, Name: ${this.name}, Price: $${this.price.toFixed(2)}`;
  }

  // getPriceWithTax is declared abstract, meaning each subclass must provide its own implementation
  abstract getPriceWithTax(): number;
}
```

### 🧠 Concept Recap:
- **`abstract class`**: Cannot be instantiated directly. It's used only as a base class.
- **`abstract method`**: Must be implemented by subclasses (`PhysicalProduct` and `DigitalProduct`).

## 🧩 **Step 5: Create the PhysicalProduct Subclass**

### 🔧 Goal
Extend the `Product` base class to model physical items with a specific tax rate and weight.

### 📄 File: `src/models/PhysicalProduct.ts`

Here’s the implementation with detailed comments:

```ts
// src/models/PhysicalProduct.ts

import { Product } from './Product';

// The PhysicalProduct class represents tangible goods (e.g., electronics, apparel).
// It extends the Product base class and adds a weight property.

export class PhysicalProduct extends Product {
  constructor(
    sku: string,
    name: string,
    price: number,
    private weight: number  // Weight in kilograms (stored privately)
  ) {
    super(sku, name, price); // Calls the parent constructor
  }

  // Returns the weight as a formatted string (e.g., "2.5 kg")
  get formattedWeight(): string {
    return `${this.weight} kg`;
  }

  // Calculates final price including 10% tax for physical items
  getPriceWithTax(): number {
    const taxRate = 0.1;
    return this.price + this.price * taxRate;
  }

  // Optionally override displayDetails to include weight
  displayDetails(): string {
    return `${super.displayDetails()}, Weight: ${this.formattedWeight}`;
  }
}
```

### 🧠 Concepts Highlighted:
- **`private`**: Limits property access to within the class.
- **`super()`**: Calls the base class constructor.
- **Override methods**: Customize base class behavior.

```ts
// src/models/DigitalProduct.ts

import { Product } from './Product';

// DigitalProduct models intangible goods like e-books or licenses.
// These products are not taxed and include a file size attribute.

export class DigitalProduct extends Product {
  constructor(
    sku: string,
    name: string,
    price: number,
    private fileSize: number // File size in megabytes (MB)
  ) {
    super(sku, name, price); // Inherit from Product
  }

  // Returns the file size as a formatted string (e.g., "500 MB")
  get formattedFileSize(): string {
    return `${this.fileSize} MB`;
  }

  // No tax applied to digital products
  getPriceWithTax(): number {
    return this.price;
  }

  // Override to include file size in the product details
  displayDetails(): string {
    return `${super.displayDetails()}, File Size: ${this.formattedFileSize}`;
  }
}
```

### 🧠 Key Concepts:
- **Overriding**: Like with `PhysicalProduct`, we override methods to tailor the behavior.
- **Encapsulation**: We use `private` for fileSize and provide a formatted getter.

## 🧩 **Step 7: Create the Tax Calculator Utility Module**

### 📄 File: `src/utils/taxCalculator.ts`

```ts
// src/utils/taxCalculator.ts

import { Product } from '../models/Product';

// This utility function calculates and returns the final price of a product,
// relying on polymorphism — it doesn't care if it's a physical or digital product.

export function calculateTax(product: Product): number {

}
```

### 🧠 Why This Works:
- **Polymorphism**: The function works with any `Product` instance.
- **Encapsulation**: Keeps tax logic separate and maintainable.

Create sample product instances and demonstrate polymorphism by looping through them to show details and tax-inclusive prices.

```ts
import { PhysicalProduct } from './models/PhysicalProduct';
import { DigitalProduct } from './models/DigitalProduct';
import { calculateTax } from './utils/taxCalculator';

// Create an array of products of different types
const inventory: Product[] = [
  new PhysicalProduct('PH001', 'Wireless Headphones', 120.00, 0.5),
  new DigitalProduct('DG001', 'E-Book: Learn TypeScript', 19.99, 15),
  new PhysicalProduct('PH002', 'Smartphone', 699.00, 0.4),
  new DigitalProduct('DG002', 'Software License', 249.00, 0.1)
];

// Loop through each product, display details and price including tax
```

- **Polymorphism**: Allows different classes to define their own versions of the same method.

#### 2. **TypeScript Features**
- **Type Annotations**: Makes the code safer and easier to understand.

## 🔥 Challenge 2: Sorting Products by Name or Price

### 🎯 Goal
Create a reusable utility that allows sorting an array of products by:
- 🏷️ **Name** (alphabetically)
- 💰 **Price** (ascending or descending)

---

### ✅ Step 1: Create the Sorting Utility

📄 File: `src/utils/sortProducts.ts`

```ts
// src/utils/sortProducts.ts

import { Product } from '../models/Product';

// This function sorts an array of Product objects
// - by: property to sort ('name' or 'price')
// - order: direction ('asc' or 'desc')
export function sortProducts(
  products: Product[],
  by: 'name' | 'price',
  order: 'asc' | 'desc' = 'asc'
): Product[] {
  return products.sort((a, b) => {
    let valueA = a[by];
    let valueB = b[by];

    if (typeof valueA === 'string') {
      // Sort names alphabetically
      const comparison = (valueA as string).localeCompare(valueB as string);
      return order === 'asc' ? comparison : -comparison;
    } else if (typeof valueA === 'number') {
      // Sort prices numerically
      return order === 'asc' ? valueA - (valueB as number) : (valueB as number) - valueA;
    }

    return 0;
  });
}
```

```ts
import { Product } from './Product';
import { DiscountableProduct } from './DiscountableProduct';

// Represents a tangible product with weight.
// Implements DiscountableProduct so we can apply percentage discounts.
export class PhysicalProduct extends Product implements DiscountableProduct {
  constructor(
    sku: string,
    name: string,
    price: number,
    private weight: number
  ) {
    super(sku, name, price);
  }

  // Returns formatted weight like "2.5 kg"
  get formattedWeight(): string {
    return `${this.weight} kg`;
  }

  // Automatically apply a 10% discount if the product is heavy
  getPriceWithTax(): number {
    const basePrice = this.weight > 1 ? this.price * 0.9 : this.price; // 10% off if > 1kg
    const taxRate = 0.1;
    return basePrice + basePrice * taxRate;
  }

  // Standard product details including weight
  displayDetails(): string {
    return `${super.displayDetails()}, Weight: ${this.formattedWeight}`;
  }

  // Manual discount (interface method)
  applyDiscount(percentage: number): void {
    this.price = this.price * (1 - percentage / 100);
  }
}
```

```ts
// src/models/PhysicalProduct.ts

import { Product } from './Product';
import { DiscountableProduct } from './DiscountableProduct'; // Interface for discount support

// PhysicalProduct represents tangible items like electronics or tools.
// It extends the base Product class and implements discounting logic.
export class PhysicalProduct extends Product implements DiscountableProduct {
  constructor(
    sku: string,
    name: string,
    price: number,
    private weight: number // Weight in kilograms
  ) {
    super(sku, name, price); // Inherit from Product constructor
  }

  // Return weight in a friendly format like "2.5 kg"
  get formattedWeight(): string {
    return `${this.weight} kg`;
  }

  // Calculates price including tax (10%),
  // and applies automatic 10% discount if the product is heavier than 1kg.
  getPriceWithTax(): number {
    const isHeavy = this.weight > 1;
    const discountedPrice = isHeavy ? this.price * 0.9 : this.price; // 10% off for heavy products
    const taxRate = 0.1;
    return discountedPrice + discountedPrice * taxRate;
  }

  // Show full product details including formatted weight
  displayDetails(): string {
    return `${super.displayDetails()}, Weight: ${this.formattedWeight}`;
  }

  // Apply a manual discount (e.g., 10%) as required by DiscountableProduct interface
  applyDiscount(percentage: number): void {
    this.price = this.price * (1 - percentage / 100);
  }
}
```

```ts
import './style.css';

interface Product {
  sku: string;
  name: string;
  price: number;
  type: string;
}

let inventory: Product[] = [];
let showDetailed = false;

document.addEventListener('DOMContentLoaded', () => {
  const form = document.querySelector('form')!;
  const skuInput = document.getElementById('sku') as HTMLInputElement;
  const nameInput = document.getElementById('name') as HTMLInputElement;
  const priceInput = document.getElementById('price') as HTMLInputElement;
  const typeSelect = document.getElementById('type') as HTMLSelectElement;
  const inventoryDiv = document.getElementById('inventory')!;

  const renderInventory = () => {
    inventoryDiv.innerHTML = '';

    if (showDetailed) {
      const table = document.createElement('table');
      table.className = 'min-w-full table-auto border-collapse';
      table.innerHTML = `
        <thead>
          <tr class="bg-gray-100 text-left">
            <th class="px-4 py-2 border">SKU</th>
            <th class="px-4 py-2 border">Name</th>
            <th class="px-4 py-2 border">Type</th>
            <th class="px-4 py-2 border">Price</th>
          </tr>
        </thead>
        <tbody>
          ${inventory.map(p => `
            <tr class="border-t">
              <td class="px-4 py-2 border">${p.sku}</td>
              <td class="px-4 py-2 border">${p.name}</td>
              <td class="px-4 py-2 border">${p.type}</td>
              <td class="px-4 py-2 border">$${p.price.toFixed(2)}</td>
            </tr>
          `).join('')}
        </tbody>
      `;
      inventoryDiv.appendChild(table);
    } else {
      inventory.forEach(p => {
        const card = document.createElement('div');
        card.className = 'p-3 border rounded bg-white shadow';
        card.innerHTML = `
          <p class="font-semibold">${p.name} (${p.type})</p>
          <p class="text-sm text-gray-600">SKU: ${p.sku}</p>
          <p class="text-sm">Price: $${p.price.toFixed(2)}</p>
        `;
        inventoryDiv.appendChild(card);
      });
    }
  };

  form.addEventListener('submit', (e) => {
    e.preventDefault();

    const sku = skuInput.value.trim();
    const name = nameInput.value.trim();
    const price = parseFloat(priceInput.value);
    const type = typeSelect.value;

    if (!sku || !name || isNaN(price)) {
      alert('Please fill out all fields correctly.');
      return;
    }

    inventory.push({ sku, name, price, type });
    renderInventory();
    form.reset();
  });

  document.getElementById('sortName')?.addEventListener('click', () => {
    inventory.sort((a, b) => a.name.localeCompare(b.name));
    renderInventory();
  });

  document.getElementById('sortPrice')?.addEventListener('click', () => {
    inventory.sort((a, b) => a.price - b.price);
    renderInventory();
  });

  document.getElementById('toggleView')?.addEventListener('click', () => {
    showDetailed = !showDetailed;
    const toggleBtn = document.getElementById('toggleView');
    if (toggleBtn) {
      toggleBtn.textContent = showDetailed ? 'Show Simple View' : 'Show Details';
    }
    renderInventory();
  });
});
```
