#### 🔄 Step 2: **Extract HTML and CSS**

In the CodePen editor:
- Copy the **HTML** (this will become your JSX)
- Copy the **CSS**

---

#### 🔧 Step 3: **Adapt HTML into JSX**
React has a few syntax rules you’ll need to follow:
| Original HTML | React JSX |
|---------------|-----------|
| `class`       | `className` |
| `for`         | `htmlFor` |
| Close tags like `<img>` | `<img />` |
| Wrap everything in a single parent | `<div> ... </div>` |

So if you copied this HTML:

```html
<div class="card">
  <img src="coffee.jpg" alt="Coffee" />
  <h2>Cappuccino</h2>
  <p>Rich and creamy blend</p>
</div>
```

You'd convert it into this JSX:

```jsx
<div className="card">
  <img src="coffee.jpg" alt="Coffee" />
  <h2>Cappuccino</h2>
  <p>Rich and creamy blend</p>
</div>
```

---

#### 🎨 Step 4: **Paste CSS into Your Project**
You can do this in several ways depending on how you manage styles:

##### Option A: **CSS file (recommended for small projects)**
1. Create a CSS file like `CoffeeCard.css`
2. Paste the styles from CodePen
3. Import it into your component:
```js
import './CoffeeCard.css';
```

##### Option B: **CSS Modules or Styled Components**
If your project is already using those, let me know and I’ll show you how to convert accordingly.

---

#### 🛠 Step 5: **Replace Your Simple Card Component**
Example `CoffeeCard.jsx` component:

```jsx
import React from 'react';
import './CoffeeCard.css';

const CoffeeCard = ({ name, description, image }) => {
  return (
    <div className="card">
      <img src={image} alt={name} />
      <h2>{name}</h2>
      <p>{description}</p>
    </div>
  );
};

export default CoffeeCard;
```

Then render a list:

```jsx
const coffeeList = [
  {
    name: 'Espresso',
    description: 'Strong and bold',
    image: '/images/espresso.jpg',
  },
  {
    name: 'Latte',
    description: 'Smooth and milky',
    image: '/images/latte.jpg',
  },
  // etc.
];

{coffeeList.map((coffee, index) => (
  <CoffeeCard key={index} {...coffee} />
))}
```
