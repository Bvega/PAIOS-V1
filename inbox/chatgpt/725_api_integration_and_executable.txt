ASSISTANT:
Here’s a simple, step-by-step guide to help you create and use the AI API Switcher GUI application:

---

### **What We'll Achieve**
We’ll build a small application that:
1. Lets you securely store API keys for OpenAI and Anthropic.
2. Allows you to choose between these APIs and send chat messages.
3. Displays responses from the APIs in a chat window.

---

### **Step 1: Install Required Software**
You need Python installed on your computer. If it isn’t already installed:
1. Go to the [Python website](https://www.python.org/).
2. Download and install the latest version.
3. During installation, check the box to "Add Python to PATH."

Now, install the required libraries by typing the following commands in your terminal or command prompt:

```bash
pip install PyQt6 requests keyring
```

- **PyQt6**: This creates the GUI (the visual part of the application).
- **Requests**: This sends messages to the APIs.
- **Keyring**: This stores your API keys securely.

---

### **Step 2: Create the Python Script**
1. Open a text editor or an Integrated Development Environment (IDE) like VS Code or PyCharm.
2. Copy the Python script below into a new file.
3. Save the file as `ai_api_switcher.py`.

Here’s the script:

```python
import sys
from PyQt6.QtWidgets import (
    QApplication, QMainWindow, QVBoxLayout, QHBoxLayout, QWidget, QLabel,
    QLineEdit, QPushButton, QTextBrowser, QComboBox, QMessageBox
)
import requests
import keyring

class AIAPISwitcher(QMainWindow):
    def __init__(self):
        super().__init__()
        self.setWindowTitle("AI API Switcher")
        self.setGeometry(100, 100, 600, 400)
        self.initUI()

    def initUI(self):
        main_layout = QVBoxLayout()

        # API Key Management
        api_key_layout = QHBoxLayout()
        self.api_key_input = QLineEdit(self)
        self.api_key_input.setPlaceholderText("Enter API Key")
        self.api_key_input.setEchoMode(QLineEdit.EchoMode.Password)
        save_key_button = QPushButton("Save API Key")
        save_key_button.clicked.connect(self.save_api_key)
        api_key_layout.addWidget(QLabel("API Key:"))
        api_key_layout.addWidget(self.api_key_input)
        api_key_layout.addWidget(save_key_button)
        main_layout.addLayout(api_key_layout)

        # Dropdown for API Selection
        self.api_dropdown = QComboBox(self)
        self.api_dropdown.addItems(["OpenAI (ChatGPT)", "Anthropic (Claude)"])
        main_layout.addWidget(self.api_dropdown)

        # Chat Display Area
        self.chat_display = QTextBrowser(self)
        self.chat_display.setPlaceholderText("Chat will appear here...")
        main_layout.addWidget(self.chat_display)

        # Message Input and Send Button
        input_layout = QHBoxLayout()
        self.message_input = QLineEdit(self)
        self.message_input.setPlaceholderText("Type your message...")
        send_button = QPushButton("Send")
        send_button.clicked.connect(self.send_message)
        input_layout.addWidget(self.message_input)
        input_layout.addWidget(send_button)
        main_layout.addLayout(input_layout)

        # Central Widget
        central_widget = QWidget()
        central_widget.setLayout(main_layout)
        self.setCentralWidget(central_widget)

    def save_api_key(self):
        """Save the API key securely."""
        api_key = self.api_key_input.text().strip()
        selected_api = self.api_dropdown.currentText()

        if api_key:
            keyring.set_password("ai_api_switcher", selected_api, api_key)
            QMessageBox.information(self, "Success", f"API key saved for {selected_api}.")
            self.api_key_input.clear()
        else:
            QMessageBox.warning(self, "Error", "API key cannot be empty.")

    def send_message(self):
        """Send a message to the selected API."""
        message = self.message_input.text().strip()
        if not message:
            QMessageBox.warning(self, "Error", "Message cannot be empty.")
            return

        selected_api = self.api_dropdown.currentText()
        api_key = keyring.get_password("ai_api_switcher", selected_api)

        if not api_key:
            QMessageBox.warning(self, "Error", f"No API key saved for {selected_api}.")
            return

        self.chat_display.append(f"User: {message}")
        self.message_input.clear()

        try:
            if selected_api == "OpenAI (ChatGPT)":
                response = self.call_openai_api(api_key, message)
            elif selected_api == "Anthropic (Claude)":
                response = self.call_anthropic_api(api_key, message)
            else:
                raise ValueError("Invalid API selection.")

            self.chat_display.append(f"{selected_api}: {response}")

        except Exception as e:
            QMessageBox.critical(self, "Error", str(e))

    def call_openai_api(self, api_key, message):
        """Call the OpenAI API."""
        url = "https://api.openai.com/v1/chat/completions"
        headers = {
            "Authorization": f"Bearer {api_key}",
            "Content-Type": "application/json"
        }
        payload = {
            "model": "gpt-4o-mini",
            "messages": [
                {"role": "system", "content": "You are a helpful assistant."},
                {"role": "user", "content": message}
            ]
        }

        response = requests.post(url, headers=headers, json=payload)
        response.raise_for_status()
        return response.json()["choices"][0]["message"]["content"]

    def call_anthropic_api(self, api_key, message):
        """Call the Anthropic API."""
        url = "https://api.anthropic.com/v1/messages/create"
        headers = {
            "Authorization": f"Bearer {api_key}",
            "Content-Type": "application/json"
        }
        payload = {
            "model": "claude-3-5-sonnet-20241022",
            "max_tokens": 1000,
            "temperature": 0,
            "system": "You are a world-class poet. Respond only with short poems.",
            "messages": [
                {"role": "user", "content": message}
            ]
        }

        response = requests.post(url, headers=headers, json=payload)
        response.raise_for_status()
        return response.json()["completion"]

if __name__ == "__main__":
    app = QApplication(sys.argv)
    window = AIAPISwitcher()
    window.show()
    sys.exit(app.exec())
```

Here's a step-by-step guide to test the Anthropic API using **Postman**:

---

### **Step 1: Open Postman**
1. Download and install [Postman](https://www.postman.com/) if you haven’t already.
2. Open the Postman application.

---

### **Step 2: Create a New Request**
1. Click the **"New"** button in the top-left corner.
2. Select **"Request"** and give it a name, e.g., "Test Anthropic API".
3. Choose a collection or create a new one to save the request, then click **Save**.

---

### **Step 3: Configure the Request**
1. **Set the Method** to **POST**.
2. **Enter the URL**: 
   ```plaintext
   https://api.anthropic.com/v1/messages/create
   ```

---

### **Step 4: Add Headers**
1. Go to the **Headers** tab.
2. Add the following headers:
   - **Key**: `Authorization`, **Value**: `Bearer YOUR_API_KEY`
     - Replace `YOUR_API_KEY` with your actual Anthropic API key.
   - **Key**: `Content-Type`, **Value**: `application/json`

---

### **Step 5: Add the Body**
1. Go to the **Body** tab.
2. Select **raw** and **JSON** (application/json) as the input format.
3. Add the following JSON payload:
   ```json
   {
       "model": "claude-3-5-sonnet-20241022",
       "max_tokens": 1000,
       "temperature": 0,
       "system": "You are a world-class poet. Respond only with short poems.",
       "messages": [
           {"role": "user", "content": "Why is the ocean salty?"}
       ]
   }
   ```

---

### **Step 6: Send the Request**
1. Click the **Send** button.
2. Postman will send the request to the Anthropic API.

---

### **Step 7: Analyze the Response**
1. If the request is successful:
   - You’ll see a **200 OK** status and the response in JSON format in the bottom panel. The response might look like:
     ```json
     {
         "completion": "The ocean's salty brine, a tale of time and design..."
     }
     ```

2. If there’s an error:
   - You’ll see an error status (e.g., `404`, `401`, or `403`).
   - Check the error message in the response body for details.

---

### **Step 8: Troubleshooting**
- **401 Unauthorized**: The API key is invalid. Recheck your API key and permissions.
- **404 Not Found**: The endpoint URL is incorrect. Confirm the correct endpoint in the Anthropic API documentation.
- **400 Bad Request**: The payload format might be wrong. Ensure the JSON structure is correct.
- **403 Forbidden**: The API key might lack the necessary permissions for this endpoint.
