why do we use span in HTML

--------------------------------------------------------------------------------

ASSISTANT:
We use `<span>` in HTML for **inline grouping of text or elements**—especially when we want to apply **styling** or **JavaScript functionality** to a specific portion of content without disrupting the flow of the text.

### Key Uses of `<span>`:

1. **Styling Inline Content:**
   Apply CSS styles to part of a sentence or phrase.
   ```html
   <p>This is a <span style="color: red;">highlighted</span> word.</p>
   ```

2. **Targeting with JavaScript:**
   Hook into small sections of content for dynamic changes.
   ```html
   <span id="username">Guest</span>
   ```

3. **Semantic Markup (with classes or IDs):**
   Categorize parts of text semantically using classes or IDs without changing the layout.
   ```html
   <span class="important">Don't forget this!</span>
   ```

### Important Notes:
- `<span>` is an **inline** element (doesn’t break lines).
- It has **no visual effect** by itself—only used when combined with CSS/JS.
- It’s the opposite of `<div>`, which is used for **block-level grouping**.
