**IMPLEMENTACIÓN DE APLICACIÓN WEB PARA CITAS DE VERIFICACIÓN DE CONTENEDORES**  
**VALIDACIONES DE LA APLICACIÓN**  

Se requiere implementar un sistema con interfaz WEB para programar las Citas de Verificación de Contenedores en SSP.

**EL OBJETIVO DE ESTA APLICACIÓN SERÁ:**  
Coordinar de manera más eficiente la logística, los procesos y las tareas involucradas en las verificaciones de contenedores realizadas en el puerto.

**LA APLICACIÓN REALIZARÁ LAS SIGUIENTES ACCIONES:**  
1. La aplicación utilizará el campo #BL para iniciar el proceso. El consignatario solicitará una cita de verificación a través del número de BL.  
2. La aplicación debe interactuar con el sistema CBS y traer los siguientes campos: BL, Número de Contenedor, Tipo de Sello, Nombre del Buque, Consignatario y Tipo de Carga.  
3. El sistema debe validar que el contenedor consultado no tenga pagos pendientes en SSP.  
4. Se podrán agendar citas de lunes a viernes de 8:00 AM a 5:00 PM, y los sábados y domingos de 8:00 AM a 12:00 PM. El área de Logística definirá los días festivos no laborables.  
5. La aplicación enviará dos reportes, uno a las 12:00 PM y otro a las 9:00 AM. Estos reportes contendrán las citas programadas para el día siguiente.  
6. La aplicación enviará una confirmación de la cita agendada al consignatario.  
7. El área de Logística deberá definir los usuarios internos o externos que recibirán el reporte con las citas agendadas en el sistema.
