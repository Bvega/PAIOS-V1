Midjourney Image Generation
Overview

Midjourney turns text prompts into unique AI-generated images via both a web app and Discord bot interface.

Users must subscribe to a plan before generating images; images are created on the “Create” page via the Imagine bar.

Workflow Steps

Log in at midjourney.com and pick your desired plan.

Type text prompts in the Imagine bar to produce images (default output: four variants per prompt).

Modify images with “upscale,” “variation,” and “re-roll” buttons to refine results.

Prompt Structure

Base command: /imagine prompt: [subtitle or description] [--parameters].

Parameters allow enhanced control and customization:

--ar (aspect ratio), e.g., --ar 3:2

--c (chaos for randomness), e.g., --c 70

--seed (deterministic re-creation), e.g., --seed 1000

--no (exclude objects/concepts), e.g., --no tree.

Image Prompting

Users can upload or reference images to guide style, content, or composition.

Style, Omni, and Character references control consistency and matching between images, supporting advanced workflows (e.g., animation, character-driven scenes).

Controls & Personalization

Users may personalize their profile for unique style consistency, create folders and moodboards, change mode and generation speed (e.g., Turbo, Draft), and organize outputs by project.

Midjourney Video Generation
Overview

Video generation transforms a still image (created in Midjourney or uploaded) into a 5-second animation using web tools only.

Discord-based image generation must be continued on the website for video creation.

Workflow Steps

Access the “Animate Image” feature under Creation Actions for images in your gallery, or use the shortcut button from the Create page.

Optionally upload and select any external image as a starting frame, adhering to Community Guidelines.

Use the Imagine bar to adjust or enter descriptive prompts before animation.

Click “Auto” to generate a video, or “Loop” to generate one with matching start/end frames.

Control video motion parameters (‘Low Motion’, ‘High Motion’) via the Motion menu section.

Video Parameters

Original image creation parameters are mostly disregarded during video generation—prompt-based changes dominate.

Basic and advanced modes provide different levels of control, emphasizing style, coherence, and movement.

API, Modification, and Advanced Tools
Controls

Comprehensive parameter list provided—users can leverage multi-prompt, weighted prompts, chaos, pan, and “no” for granular output control.

Website and Discord have independent command lists, but all major features transition to the web for video and organization tasks.

API Access

Direct, official API access is not public yet; several unofficial APIs exist for automation.

Example endpoint: /imagine for images, /describe for prompt generation based on images.

Organization and Collaboration
Creation management: Organize images/videos with folders, download features, and filtering tools.

Multi-user collaboration: Options to share, showcase, and chat with community members and daily themed galleries.
