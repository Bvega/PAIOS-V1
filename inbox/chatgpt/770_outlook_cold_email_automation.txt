ASSISTANT:
Yes, we can automate this cold email process directly in Outlook and Excel using VBA macros. Below is a VBA-based workflow to send emails based on your Excel data and track the status without needing Power Automate.

### Stage 1: Setting Up the Excel VBA Script

1. **Open the VBA Editor**:
   - Press `Alt + F11` in Excel to open the VBA editor.
   - In the editor, go to **Insert** > **Module** to create a new module where the VBA script will be placed.

2. **Create the VBA Code for Email Automation**:
   - Copy and paste the following VBA code into the module. This script reads the data from your Excel sheet, sends an email via Outlook for each row with "Not Sent" status, and updates the status to "Sent."

```vba
Sub SendColdEmails()
    Dim OutlookApp As Object
    Dim EmailItem As Object
    Dim ws As Worksheet
    Dim lastRow As Long
    Dim i As Long
    
    ' Set references
    Set ws = ThisWorkbook.Sheets("Sheet1") ' Replace with your sheet name
    Set OutlookApp = CreateObject("Outlook.Application")
    
    ' Find the last row with data in the sheet
    lastRow = ws.Cells(ws.Rows.Count, "A").End(xlUp).Row
    
    ' Loop through each row
    For i = 2 To lastRow ' Start from 2 to skip header row
        ' Check if status is "Not Sent"
        If ws.Cells(i, 4).Value = "Not Sent" Then
            ' Create the email
            Set EmailItem = OutlookApp.CreateItem(0)
            With EmailItem
                .To = ws.Cells(i, 1).Value ' Email Address
                .Subject = "Introduction from " & ws.Cells(i, 3).Value ' Use Company Name in Subject
                .Body = "Hello " & ws.Cells(i, 2).Value & "," & vbCrLf & vbCrLf & _
                        "I would like to introduce myself..." ' Add your email body text here
                .Send ' Sends the email
            End With
            
            ' Update the status and date in Excel
            ws.Cells(i, 4).Value = "Sent"
            ws.Cells(i, 5).Value = Date ' Date Sent column
        End If
    Next i
    
    ' Clean up
    Set EmailItem = Nothing
    Set OutlookApp = Nothing
    MsgBox "Cold emails sent and statuses updated."
End Sub
```

3. **Modify the Code as Needed**:
   - Adjust the email body content in `.Body` to fit your message.
   - Ensure that the worksheet name in `Set ws = ThisWorkbook.Sheets("Sheet1")` matches your actual sheet name.

4. **Run the Macro**:
   - Close the VBA editor and save your Excel file as a **Macro-Enabled Workbook** (`.xlsm`).
   - Press `Alt + F8`, select `SendColdEmails`, and run the macro to send emails to recipients with "Not Sent" status.

### Stage 2: Setting Up the Follow-up Workflow (Optional)

To add follow-up emails, we can set up another VBA macro that:
   - Checks for rows with a **Date Sent** older than a week.
   - Sends a follow-up email.
   - Updates a **Follow-up Status** column to "Followed Up."

```vba
Private WithEvents inboxItems As Outlook.Items

    Dim ns As Outlook.Namespace
    Set ns = Application.GetNamespace("MAPI")
    Set inboxItems = ns.GetDefaultFolder(olFolderInbox).Items

Private Sub inboxItems_ItemAdd(ByVal Item As Object)
    ' Runs whenever a new item is added to the Inbox
    Dim mail As Outlook.MailItem
    Dim targetFolder As Outlook.Folder
    Dim ns As Outlook.Namespace
    Dim idPattern As String

            If targetFolder Is Nothing Then
```

```vba
Private WithEvents inboxItems As Outlook.Items

Private Sub Application_Startup()
    ' Set up the inbox to monitor incoming items
    Dim ns As Outlook.Namespace
    Set ns = Application.GetNamespace("MAPI")
    Set inboxItems = ns.GetDefaultFolder(olFolderInbox).Items
End Sub

Private Sub inboxItems_ItemAdd(ByVal Item As Object)
    ' Runs whenever a new item is added to the Inbox
    Dim mail As Outlook.MailItem
    Dim targetFolder As Outlook.Folder
    Dim ns As Outlook.Namespace
    Dim idPattern As String
    Dim match As Object

    If TypeOf Item Is Outlook.MailItem Then
        Set mail = Item
        
        ' Use regular expression to find ProspectID in the subject or body
        idPattern = "ProspectID(\d+)" ' Adjust pattern as needed
        Set match = CreateObject("VBScript.RegExp")
        match.IgnoreCase = True
        match.Pattern = idPattern

        If match.Test(mail.Subject) Or match.Test(mail.Body) Then
            ' Extract ProspectID
            Dim prospectID As String
            prospectID = match.Execute(mail.Subject & mail.Body)(0).SubMatches(0)
            
            ' Define the target folder based on ProspectID
            Set ns = Application.GetNamespace("MAPI")
            On Error Resume Next
            Set targetFolder = ns.GetDefaultFolder(olFolderInbox).Folders("Prospect Response").Folders(prospectID)
            On Error GoTo 0
            
            ' Create the folder if it doesn't exist
            If targetFolder Is Nothing Then
                Set targetFolder = ns.GetDefaultFolder(olFolderInbox).Folders("Prospect Response").Folders.Add(prospectID)
            End If
            
            ' Move the email to the specific folder
            mail.Move targetFolder
        End If
    End If
End Sub
```
